      Здесь описываются файлы,входящие в состав системы: TB.EXE - все-
 объемлющая разработка оборудование/редактор/компилятор/библиотека  ра-
 бочих программ.Для запуска программы наберите на пульте TB+Пуск.
      .BAS - эталонные программы Turbo-Basic.
      TBHELP.TBH -  файл,содержащий справочную информацию,которую можно
 затребовать,находясь в Turbo-Basic.
      README.COM - программа для считывания файла README.  README - это
 текстовый файл,содержащий  более  насущную  информацию,чем  та,которая
 имеется в данном учебном пособии.
