Синтаксис:   $SEGMENT

Комментарии: $SEGMENT  используется  для разбивки Вашей исходной прог-
              раммы,когда компилятор сообщает о превышении размеров ко-
              дового сегмента (64К): ERROR 409 Segment Overlow. Наличие
              $SEGMENT приводит к тому,что выполнение операторов управ-
              ления  потоком  (GOSUB,GOTO),переходящих  границы сегмен-
              та,инициализирует больше места в стеке и  занимает больше
              времени на выполнение. В одной программе может содержать-
              ся до 16 сегментов.Размеры  каждого  сегмента,разделенные
              косой  чертой,указываются после компиляции в окне Messing
              Turbo-Basic. Примечание:$SEGMENT не должен использоваться
              внутри структурного блока,например:FOR/NEXT,DO/LOOP,WHILE
              /WEND.

Пример:      SUB Proc1
                PRINT "This is a dummy procedure"
              SUB END

              $SEGMENT

              SUB Proc2
                PRINT "This is another dummy procedure"
              END SUB

              $SEGMENT

              SUB Proc3
                PRINT "This is another dummy procedure"
              END SUB

              CALL Proc1
              CALL Proc2
              CALL Proc3
              END
