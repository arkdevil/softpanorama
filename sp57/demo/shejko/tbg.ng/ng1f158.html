Синтаксис:$COMn size Комментарии: n означает адаптер связи (1 или 2).
              size - целочисленная постоянная,определяющая емкость
              буфера для этого адаптера (от 0 до 32.767).Умолчание сос-
              тавляет 256.Значение умолчания может устанавливаться и
              храниться в меню Option.Значения умолчания могут изме-
              няться метаопрератором в программе.

Ограничения:Turbo-Basic может резервировать буфера различной емкости
              под каждый из двух адаптеров (в отличие от интерпретирую-
              щего Basic).

Пример:     $COM1 1024
                ON COM(1) GOSUB GetComInput
                COM(1) ON
                OPEN "COM1" AS #1
                PRINT "Press any key to terminate the program..."
                WHILE NOT INSTAT
                  LOCATE 2,1
                  PRINT TIMES$
                WEND
                END

                GetComInput:
                  INPUT #1,ComPortInput(HeadPtr%)
                  RETURN

                EndOfInput:
                  PRINT "Reached the end of input..."
                  END
