 Синтаксис:   y=PEEK (offset)

 Комментарии: offset - численное выражение в пределах от  0  до  65535,
              представляющее  16-битовое  смещение  в текущем сегменте,
              определенного с помощью  оператора  DEF  SEG.Возвращаемое
              значение  может  находиться в пределах от 0 до 255.PEEK и
              оператор POKE представляют собой команды  самого  низкого
              уровня работы в Turbo-Basic.
See Also: DEF SEG POKE
