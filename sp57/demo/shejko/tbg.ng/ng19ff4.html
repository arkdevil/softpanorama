 оператором KEY OFF.
      При монохромном адаптере оператор  COLOR  может  применяться  для
 создания специальных эффектов,наприме,увеличенная интенсивность свече-
 ния,мерцание и/или подчеркивание текста.В CGA/EGA адаптерах этот  опе-
 ратор служит для выбора цвета выводимых символов.

 'страниц',которые обрабатываются с помощью оператора PRINT.
