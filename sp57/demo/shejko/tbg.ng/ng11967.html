      Числовые операторы разбиваются на три  основные  группы:арифмети-
 ческие,относительные,логические.
      Арифметические операторы выполняют нормальные математические опе-
 рации.Их перечень приведен в таблице 4-2.


             Таблица 4-2.Арифметические операторы.
 ---------------------------------------------------------------------
 Оператор           Действия                       Пример
 ---------------------------------------------------------------------
          Возведение в степень                    104
 -         Отрицание                               -16
 *,/       Умножение,деление с плавающей запятой   45*19,45/19
 \         Целочисленное деление                   45\19
 MOD       Деление по модулю                       45 MOD 19
 +,-       Сложение,вычитание                      45+19,45-19
 ---------------------------------------------------------------------

 Рассмотрим подробнее два оператора из таблицы:
 Целочисленное деление округляет свой результат до целого числа:

      5\2=2
      9\10=0

 Оператор MOD в качестве результата возвращает остаток деления:

      5 MOD 2=1
      9 MOD 10=9

      Относительные операторы  позволяют  Вам  сравнивать значения двух
 чисел или двух цепочек для получения Булева результата TRUE или FALSE.
 Результат  сравнения  получает  значение -1 (TRUE) и 0 (FALSE) и могут
 использоваться в любых числовых выражениях или операторах с  конструк-
 цией типа IF:

      PRINT 5>6 распечатает 0
      PRINT 5<6 распечатает -1
      PRINT (5<6)15 распечатает -15
      a=(b>c)/13
      IF a>b THEN GOTO 60
                                 .

 В таблице 4-3 приведен перечень относительных операторов.


              Таблица 4-3.Относительные операторы.
 ---------------------------------------------------------------------
   Оператор                    Соотношение                 Пример
 ---------------------------------------------------------------------
      =                        Равенство                     x=y
      <>                       Неравенство                   x<>y
      <                        Меньше,чем                    x<y
      >                        Дольше,чем                    x>y
      <=                       Меньше,чем или равно          x<=y
      >=                       Больше,чем или равно          x>=y
 ---------------------------------------------------------------------

 Когда арифметические и относительные операторы комбинируются в выраже-
 ние,то  арифметические операции всегда расчитываются в первую очередь.
      Логические операторы выполняют логические (Булевы)  операции  над
 целыми  числами.Используемые  совместно относительными операторами,они
 разрешают Вам проверять сразу несколько условий:

      IF uk>29 AND mon=2 THEN PRINT "Error in date"

 В таблице 4-4 дан перечень логических операторов.


                Таблица 4-4.Логические операторы
 ---------------------------------------------------------------------
    Оператор         Операция               Выполняемые действия
 ---------------------------------------------------------------------
      AND           Логическое 'И'                 x  0 0 1 1
                                                 y  0 1 0 1
                                           x AND y  0 0 0 1
      OR            Логическое 'ИЛИ'               x  0 0 1 1
                                                 y  0 1 0 1
                                            x OR y  0 1 1 1
      XOR           Сложение по модулю 2           x  0 0 1 1
                                                 y  0 1 0 1
                                           x XOR y  0 1 1 0
      EQV           Эквивалентность                x  0 0 1 1
                                                 y  0 1 0 1
                                           x EQV y  1 0 0 1
      IMP           Импликация                     x  0 0 1 1
                                                 y  0 1 0 1
                                           x IMP y  1 1 0 0
      NOT           Отрицание                      x  1 0
                                            NOT (x) 0 1
 ---------------------------------------------------------------------

 Логические операторы могут также проводить и побитовую обработку  дан-
 ных.Наиболее применимыми сдесь являются операторы AND,XOR,OR.

             Цепочки и относительные операторы.

      Turbo-Basic позволяет  Вам  сравнивать  цепочки знаков на предмет
 равенства,а также 'больше,чем' и 'менньше,чем'.
      Две цепочки  равны,если они содержат одни и те же знаки в одинко-
 вом порядке.Одна цепочка меньше другой,если ее длина меньше длины вто-
 рой  цепочки  или  она содержит символ,имеющий меньший код в ASCII,чем
 соответствующий символ второй цепочки.
