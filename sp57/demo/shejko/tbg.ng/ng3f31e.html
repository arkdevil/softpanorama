 Синтаксис:   PRINT USING format string;expression list[;]

 Комментарии: format string - строковая константа или переменная,описы-
              вающая метод форматирования информации в expression list.
               expression list   -  последовательности  числовых  и/или
              строковых выражений,разделенных точками с  запятой,пробе-
              лами или запятыми.
               Задание формата для вывода строковых значений.
               Чтобы распечатать  первые  n  значений  строки,применяют
              символ \.Turbo-Basic распечатывает  столько  символов  из
              строки,сколько пробелов включает в себя строка управления
              форматом (включая символы \).Символ ! распечатывает толь-
              ко один символ строки,а & - все символы.Например:

              a$="cats and dogs"
              PRINT USING "!";a$
              PRINT USING "\\";a$
              PRINT USING "\  \";a$
              PRINT USING "&";a$

              В результате получиться:
              c
              ca
              cats
              cats and dogs

              Форматирование числовых значений.
              Форматирующий знак  # выравнивает выводимые числовые зна-
              чения справа.Если количество разрядов  числового значения
              меньше  числа  позиций в зоне,то это значение дополняется
              слева пробелами.Если значение содержит слишком много цифр
              и  не  помещается  в  зоне,задаваемой шаблоном,то оно все
              равно выводится полностью,а перед его первой цифрой выво-
              дится  знак %.Если выводимое значение отрицательно,то вы-
              водится знак минус.Например:

              PRINT USING "####";12
              PRINT USING "####";-123
              PRINT USING "####";123456

              В результате получится:
                12
              -123
              %123456
