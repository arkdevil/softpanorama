 Синтаксис:   y=CDBL (numeric expression)

 Комментарии: CDBL   преобразует   числовую  переменную  или  выражение
              (numeric expression) в форму  удвоенной  точности.Исполь-
              зуйте  CDBL для удаления возможности целочисленного пере-
              полнения в арифметических операциях.Например:

              y#=(z%+x%)/a#

              При выполнении этого  выражения  получится  переполнение,
              если  z%+x%  болше,чем 32767.Поэтому результирующую пере-
              менную надо привести к удвоенной точности:

              y#=(CDBL(z%)+x%)/a#

 Ограничения: область выражения,отосланная на CDBL,должна быть внутри
              диапазона для чисел с удвоенной точностью.

 Пример:      x%=2000
              y%=2000
              z#=CDBL(x%)*y%
              PRINT x%,y%,z#
              END
