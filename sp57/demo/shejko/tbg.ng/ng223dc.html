 Синтаксис:   BSAVE filespec,address,length

 Комментарии: filespec определяет имя файла,в  который  будут  записаны
              данные.
              address - числовое выражение от 0  до  65535,определяющее
              смещение в текущем сегменте (оператор DEF SEG),с которого
              будет  считана  информация.   length   определяет   число
              байт,которое надо считать ( от 0 до 65535). Файлы,создан-
              ные BSAVE,могут загружаться (причем и с  другого  адреса)
              оператором  BLOAD.  Тандем этих команд часто используется
              для сохранения и загрузки изображения на  экране.Дисплей-
              ный  буфер  адаптера  монохромного  дисплея имеет в длину
              4000 байт  и  начинается  со  смещения  0  в  сегменте  &
              HB000.Дисплейный буфер для цветографического дисплея име-
              ет длину 4000 байт в текстовом режиме и 16384 байт в обо-
              их  графических  режимах;оба  начинаются  со смещения 0 в
              сегменте &HB800.

 Примечание:  перед выполнением BSAVE установите оператором DEF SEG
              нужный сегмент.

 Пример:      FOR i%=1 TO 2000
                PRINT "A";
              NEXT i%

              DEF SEG=&HB8000

              BSAVE "TEXT.PIC",0,4000
              END
See Also: BLOAD DEF SEG
