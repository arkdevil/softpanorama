 Синтаксис:   для функции      y=MTIMER
              для оператора    MTIMER

 Комментарии: MTIMER  предназначен  для измерения времени счета,главным
              образом для кратковременных операций.Она предлагает улуч-
              шенную разрешающую способность.Однако точность резко сни-
              жается приблизительно через 54 миллисек.  Функция  MTIMER
              возвращает  то количество микросекунд,  которое отсчитано
              со времени выполнения самого последнего оператора MTIMER.
              Минимальная  временная градация функции MTIMER составляет
              2 микросекунды. Оператор MTIMER сбрасывает микротаймер.

Примечание: оператор и функция MTIMER используются парами.Сначала вы-
              зывается оператор MTIMER,затем вызывают функцию и получа-
              ют результат.Если оператор MTIMER не запустит заново мик-
              ротаймер,последующие  ссылки  на   функцию   вырабатывают
              значение '0'. MTIMER и SOUND/PLAY каждый используют канал
              2 таймера 8255 и являются взаимоисключающими.
