Синтаксис:$DYNAMIC

Комментарий:$DYNAMIC не содержит аргументов и  только  объявляет тип
              массивов.Пространство  для динамических массивов резерви-
              руется во время прогона.Вы можете  использовать  оператор
              ERASE для удаления ненужных массивов.Динамические массивы
              обеспечивают более рациональное использование памяти,т.к.
              их  размерность  определяется из конкретных условий прог-
              раммы.  Массивы могут объявляться как динамические и  при
              использовании  ключевого  слова DYNAMIC в самом операторе
              DIM.  Массивы с переменными размерами,объявленные локаль-
              ными в функции или процедуре,всегда динамические.


Пример:     $DYNAMIC
                ON ERROR GOTO ErrorHandler
                PRINT FRE(-1)
                DIM BigArray(10000)
                BigArray(6666)=66
                PRINT FRE(-1)
                ERASE BigArray
                PRINT FRE(-1)
                PRINT BigArray(6666)
                END

                ErrorHandler:
                  PRINT "An error of type"ERR;
                  PRINT "has occurred at address"ERADR
                  END
