 Синтаксис:   POKE offset,value

 Комментарии: offset - смещение в текущем сегменте в пределах от  0  до
              65535.
               value -  байт,который  должен  быть  записан  по  адресу
              offset.Значение  value  должно находиться в пределах от 0
              до 255.
               Т.к. при этой операции не производится контроль абсолют-
              ного адреса,то можно испортить DOS или программу при неп-
              равильном использовании оператора.
