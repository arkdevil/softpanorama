 Синтаксис:   COMMON variable[,variable]...

 Комментарии: variable является переменной или размером массива,который
              должен быть сохранен при вызове другой программы. Массивы
              определяются включением числа размеров в  круглые  скобки
              после имени массива.
              Операторы COMMON могут появляться в любом месте  програм-
              мы,но  данная переменная может появиться в перечне только
              один раз.
              Согласующий оператор  COMMON должен появляться в вызывае-
              мой программе.Если тип и количество передаваемых перемен-
              ных  в  вызывающей и вызываемой программе не совпадает,то
              формируется ошибка прогона.

Примечания:  наименования переменных в программах,связанных COMMON,мо-
              гут различаться,но их типы должны соответствовать друг
              другу.

 Различия:    Turbo-Basic не поддерживает оператора CHAIN ALL COMMON
              из Basic.

 Пример:      DIM Array$(1)
              Array$(0)="This is the first string in the array"
              Array$(1)="This is the second string in the array"
              Integer%=12345
              LongInt&=123450000
              SinglePre!=12345.12345
              DoublePre#=1.0E+300

              COMMON Array$(),Integer%,LongInt&,SinglePre!,DoublePre#
              CHAIN "SLAVE.TBC"
              END

              'Program SLAVE
              COMMON NewArray$(),NewInteger%,NewLongInt&,NewSinglePre!_
                     NewDoublePre#
              PRINT NewArray$(0)
              PRINT NewArray$(1)
              PRINT NewArray$(),NewInteger%,NewLongInt&,NewSinglePre!_
                    NewDoublePre#
