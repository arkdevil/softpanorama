 Синтаксис:   MKDIR path

 Комментарии: path -  строковое  выражение,описывающее  каталог,который
              необходимо сформировать.  (См. Гл.4 - "Информация об име-
              нах файлов и префиксах имен файлов")
              MKDIR (сформировать каталог)формирует каталог,определямый
              префиксом имен файла.  Если попытаться  сформировать  уже
              существующий каталог,  будет иметь место ошибка 5,Illegal
              Function Call(вызов запрещенной функции).
