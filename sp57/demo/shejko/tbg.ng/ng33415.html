 Синтаксис:   y=LEN(string expression)

 Комментарии: LEN возвращает значение от 0 до 32767,представляющее со-
              бой количество знаков в строке.

 Пример:      INPUT "Enter a string: ",Dummy$
              PRINT LEN(Dummy$)
