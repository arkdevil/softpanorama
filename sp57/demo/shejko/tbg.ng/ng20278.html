Синтаксис:   $INCLUDE filespec literal

Комментарии: $INCLUDE  позволяет  во  время  компиляции текущего
              файла включать  в  его  исходный  текст  текст  другого
              файла. filespec literal представляет собой цепочку,иден-
              тифицирующую дозагружаемый файл.Механизм $INCLUDE побуждает
              компилятор  обрабатывать  включенный файл,как если бы он был
              физически в исходном тексте.Концепция рабочие файлы/  ос-
              новной  файл позволит Вам разбивать Вашу программу на от-
              дельные части.

Пример:
                    File CAREA.BAS

              PRINT "Circle area calculating program"
              PRINT "If yon find this program useful"
              PRINT "Please send $3.500 to:"
              PRINT
              PRINT "Frank Borland"
              PRINT "4585 Scotts Valley Drive"
              PRINT "Scotts Valley,CA 95066"
              PRINT "--------------------------------"
              $INCLUDE "CONST.BAS"
              INPUT "Enter radius",r
              PRINT "Area = "pi*r*r
              END


                    File CONST.BAS

              pi=ATN(1)*4
              %true=-1
              %false=0
              %maxx=319
              %many=199

              Для компилирования этой программы CAREA.BAS  должена быть
              сделана  'основным  файлом',т.к.  он  является  файлом  с
              $INCLUDE (подробнее см.  Гл3. 'Main Menu'). Независимо от
              того,какой  рабочий  файл находится в данный момент в ре-
              дакторе,компиляция программы начнется с  основного  файла
              (в  нашем  случае  - CAREA.BAS) и при достижении $INCLUDE
              компиляция приостанавливается для загрузки с диска  рабо-
              чего файла.$INCLUDE позволяют делать вложения глубиной до
              5.
