      Подпрограммы представляют  собой  традиционный  метод   разбиения
 программ Basic.Они состоят из групп операторов,заканчивающихся RETURN.
 Для вызова подпрограммы используется оператор GOSUB с  указанием метки
 первой  строки подпрограммы.После выполнения оператора RETURN управле-
 ние передается к следующему за GOSUB оператору.

      GOSUB AddMonths
      PRINT total
      END

      AddMonths:
         total=0
         FOR i=1 TO 12
           total=total+month(i)
         NEXT i
      RETURN
