 Синтаксис:   CLS

 Комментарии: в текстовом режиме CLS очищает весь экран и устанавливает
              курсор в его левый верхний угол.Цвет очистки - фоновый. В
              графическом режиме очищается экран фоновым цветом.LPR по-
              мещается в центр экрана.Если  оператор  VIEW  был  выпол-
              нен,то  CLS  очищает  лишь порт представления.Для очистки
              всего экрана используйте VIEW без аргументов,а затем CLS.

 Пример:      PRINT "This is a test of the CLS statement"
              INPUT "Press <ENTER> to clear the screen...",Dummy$
              COLOR 10,1
              CLS
              END
See Also: COLOR SCREEN VIEW
