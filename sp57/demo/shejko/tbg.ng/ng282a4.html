 Синтаксис:   DATA constant[,constant]...

 Комментарии: constant является числовой или строковой постоянной. Чис-
              ловые постоянные могут быть с плавающей запятой или целы-
              ми  числами и могут выражаться в 16-ом или 8-ом представ-
              лении.
              Строковые константы  могут не заключаться в ограничитель-
              ные кавычки,если они не содержат  внутренних разделителей
              и могут свободно смешиваться с числовыми константами:

              DATA Taco,.79,Supreme,2.29,"Pepper,lange"

              Операторы DATA  могут находиться в разных местах програм-
              мы,и каждый может  содержать  столько  постоянных,сколько
              позволяет строка.
              Во время прогона оператор READ считывает данные  из  DATA
              слева  направо и сверху вниз.При попытке считать строку в
              числовую переменную возникнет  ошибка  2  прогона,но  при
              считывании  числового  значения  в  строковую  переменную
              ошибки не будет. При попытке считывать данные за предела-
              ми  последнего  DATA выдается ошибка 4 прогона.  Оператор
              RESTORE позволяет установить указатель на нужный оператор
              DATA.

 Ограничения: в операторе DATA нельзя использовать  знак подчеркивания.
              Кроме этого,не применяйте одиночные кавычки для коммента-
              риев,т.к. Turbo-Basic воспримет Ваш комментарий как стро-
              ковую  постоянную.Тем не менее Вы можете использовать REM
              для этой цели:

              DATA cats,dogs,pigs:REM list the animals


 Пример:      SUB Process (StudentName$,StudentAverage%)
                LOCAL Total%,Value%,i%
                READ StudentName$
                FOR i%=1 TO 10
                  READ Value%
                  Total%=Total%+Value%
                NEXT i%
                StudentAverage%=Total%\10
              END SUB

              FOR Student%=1 TO 3
                CALL Process(Name$,Average%)
                PRINT Name$,Average%
              NEXT Student%

              END

              DATA M.Cissone,44,32,65,76,65,43,56,32,56,77
