      Выберете режим Edit основного меню Turbo-Basic.
 Во-первых,посмотрите на строку состояния,чтобы определить,имеете ли Вы
 режим  Insert (вставка символа в текущую позицию экрана) или Owerwrite
 (изменение символа в текущей позиции экрана).Этот режимы переключаются
 клавишей Ins (или Ctrl-V).
      Теперь напечатайте,например,Ваш адрес и нажмите ESC для  заверше-
 ния  редактирования.Для  сохранения содержимого рабочего файла выполните
 опцию Save команды File.Поскольку Вы ранее не присвоили имя этой прог-
 рамме,то по умолчанию файл будет записан под именем NONAME.BAS.
      Теперь используя опцию New сотрите то,что Вы создали  в редакторе
 и  введите  новый текст.Затем закончите редактирование и в режиме File
 основного меню выберете опцию Load с  именем  NONAME  (загрузка  файла
 NONAME).Система спрашивает ,желаете-ли Вы сохранить рабочий тест,запи-
 санный в редакторе и после Вашего ответа считывает программу  из файла
 NONAME.BAS
