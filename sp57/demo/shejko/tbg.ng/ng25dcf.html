 Синтаксис:   COM(n){ON│OFF│STOP}

 Комментарии: n определяет номер адаптера связи (1 или  2).
              COM(n) ON  разрешает  прерывание  от  адаптера связи.
              ON COM(n) определяет подпрограмму  обработки  прерывания.
              COM(n)  STOP  не  выполняет прерывание,но запоминает его.
              Использовать матаоператор $COM для изменения размера  бу-
              фера COM.

 Пример:      ON COM(1) GOSUB GetComInput
              DIM ComPortInput (5*1024)
              HeadPtr%=0:TailPtr%=0

              COM(1) ON
              $COM1 1024
              OPEN "COM1" AS#1

              PRINT "Press any key to terminate the program..."
              WHILE NOT INSTAT
                IF TailPtr%><HeadPtr% THEN
                  PRINT "COM Port input: ";ComPortInput(TailPtr%)
                  TailPtr%=TailPtr%+1
                END IF
                LOCATE 2,1
                PRINT TIME$
              WEND
              END

              GetComInput:
                INPUT #1,ComPortInput(HeadPtr%)
                HeadPtr%=HeadPtr%+1
                RETURN

              EndOfInput:
                PRINT "Reached the end of input..."
                END
See Also: ON COM(n)
