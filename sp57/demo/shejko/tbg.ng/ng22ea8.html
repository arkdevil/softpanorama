 Синтаксис:   CALL ABSOLUTE address(parameter list)

 Комментарии: address  - числовое выражение,определяющее смещение внеш-
              ней программы,к которой будет обращеНие.СЕгмент определя-
              ется  последним  оператором  DEF  SEG.Диапазон address от
              -32768 до 65535.Блок управления файлом  (см.  техническую
              документацию  на  DOS) инициируется перед CALL ABSOLUTE и
              содержит информацию о текущем состоянии процессора.Доступ
              к  этому  буферу  обеспечивается  с помощью оператора REG
              (загружает выбранный элемент буфера указанным целочислен-
              ным  значением)  и  функции REG (читает значение элемента
              буфера).Например:

              ---------------------------------
              Аргумент REG          Регистр
              ---------------------------------
              0                     Flags
              1                     AX
              2                     BX
              3                     CX
              4                     DX
              5                     SI
              6                     DI
              7                     BP
              8                     DS
              9                     ES
              ---------------------------------

              REG 2,&H1000
              PRINT REG(2)

              Программист несет ответственность за сохранение регистров
              DS,BP,SS,SP.

 Пример:      DEFINT a-z
              DIM a(100)
              DIM FillRoutine(10)
              Fillroutine(0)=&HF2FC
              Fillroutine(1)=&HCBAA

              REG 1,-1
              REG 3,101 * 2
              REG 9,VARSEG(a(0))
              REG 6,VARPTR(a(0))

              DEF SEG=VARSEG(FillRoutine(0))
              PRINT a(1)
              WHILE NOT INSTAT:WEND
              FillAdr=VARPTR(FillRoutine(0))
              CALL ABSOLUTE FillAdr (REG)

              PRINT a(1)
              END
