 Синтаксис:   OPTION BASE integer expression

 Комментарии: integer  expression  может  находиться в пределах от 0 до
              32.767.
               OPTION BASE  указывает  минимальное значение первого ин-
              декса массива.Например,описанный вслед за OPTION  BASE  1
              массив   DIM   x(20)   резервирует  элементы  (1-20).Хотя
              TurboBasic и описывает нижние пределы массивов операторам
              OPTION  BASE,мы  предлагаем более действенный метод с по-
              мощью оператора DIM.
