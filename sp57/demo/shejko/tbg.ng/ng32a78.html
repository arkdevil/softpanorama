 Синтаксис:   INPUT [;] [prompt string{;│,}] variable list

 Комментарии: prompt string (цепочка подсказки) - необязательная  стро-
              ковая постоянная,представляющая собой подсказку для поль-
              зрвателя.
              variable list  -  разделенная запятыми последовательность
              одной или нескольких строковых или  числовых  переменных.
              INPUT  присваивает  вводимые с клавиатуры данные variable
              list,причем тип вводимых данных должен сочетаться с типом
              переменных  в variable list и вводимые данные должны раз-
              деляться запятыми. Если после prompt string стоит точка с
              запятой,то после строки подсказки выводится знак вопроса.
              Если точка с запятой появляется после слова INPUT,то пос-
              ле ввода данных и нажатия ENTER курсор не перемещается на
              новую строку,а остается на прежней.

 Различие:    Turbo-Basic позволяет разделять вводимую информацию либо
              запятыми,либо пробелами,Basic - только запятыми.
