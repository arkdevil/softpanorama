      Turbo-Basic имеет так называемые машинные файлы,т.е. он может об-
 щаться с клавиатурой,дисплеем,принтером как с последовательными файла-
 ми.Каждое из устройств имеет свое зарезервированное имя файла:
..page10

 ---------------------------------------------------------------------
 Имя                                     Функция
 ---------------------------------------------------------------------
 KYBD:              Клавиатура может открываться для ввода.Считывание
                  происходит подобно использованию INKEY$
 SCRN:              Экран может открываться для вывода
 LPT1-3:            Принтеры от 1 до 3 - открываются для вывода
 COM1-2             Порты связи 1 и 2
 ---------------------------------------------------------------------

 Например:

      OPEN "SCRN:" FOR OUTPUT AS #1:PRINT #1,"Hello"
      или
      OPEN "KYBD:" FOR INPUT AS #1:INPUT #1,a$,b$
