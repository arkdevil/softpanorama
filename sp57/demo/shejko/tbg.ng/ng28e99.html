 Синтаксис:   одна строка:

              DEF FNidentifier [(argument list])=expression

              несколько строк:

              DEF FNidentifier [(argument list])
                [LOCAL variable list]
                [STATIC variable list]
                [SHARED variable list]
                .
                . statements
                .
                [EXIT DEF]
                [FNidentifier=expression]
              END DEF

 Комментарии: identifier - имя функции.  argument list (список аргумен-
              тов) - необязательный список параметров,разделенный запя-
              тыми.Параметры,используемые  в  списке  аргументов,служат
              только для определения функции и не имеют связи с  однои-
              менными  параметрами  в программе.
              При вызове функции ей можно передать один или более пара-
              метров.После   выполнения  необходимых  действий  функция
              возвращает значение,тип которого  определяется  из  имени
              функции.Поэтому функция может вызываться из любого опера-
              тора.
                    Объявление функций и ход выполнения программы: мес-
              то определения функции несущественно и не зависит от мес-
              та  вызова  функции.Компилятор 'помечает' Ваши функции (и
              процедуры),так что при выполнении программа не может слу-
              чайно  'впадать  в функцию' (в противоположность подпрог-
              раммам).  Не применяйте в функциях операторов  GOTO,GOSUB
              или  RETURN которые передают управление за пределы описа-
              ния функции.  Определения функций или процедур  не  могут
              вкладываться, однако в определениях могут содержаться вы-
              зовы функций или процедур.
                     Объявление локальных  переменных  (только для мно-
              гостроковых функций):
              для объявления локальных переменных используется оператор
              LOCAL в описании переменных,которое применяется перед ра-
              бочим телом функции.Например:

              LOCAL a%,b#,Array%()

              создает 3 локальных переменных:a%,b# и целочисленный мас-
              сив bigArray.Массив должен,таким  образом,получать  соот-
              ветствующие переменные:

              DIM DYNAMIC Array%(1000)

              По умолчанию переменные функции имеют атрибут SHARED,т.е.
              они становяться глобальными после вызова функции.
              Объявление переменных  с помощью атрибута STATIC приведет
              к тому,что переменные не будут терять свое  значение  при
              каждом  обращении  к функции,но в то же время будут изме-
              няться только в данной функции. Многострочное определение
              функции должно заканчиваться оператором END DEF.  Исполь-
              зуйте оператор EXIT DEF для возвращения из  функции, если
              Вы находитесь в теле функции,а не в ее конце.

 Пример:      $STACK &H7FFF
              Total%=1

              DEF FNFactorial%(i%)
                Total=Total*i%
                IF i%>1 THEN
                  Subb%=FNFactorial%(i%-1)
                END IF
                FNFactorial%=Total
              END DEF

              PRINT "Input the number you wish to calculate the ";

              END
