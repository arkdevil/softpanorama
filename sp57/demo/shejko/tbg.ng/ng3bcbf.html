 Синтаксис:   PALETTE [attribute,color)
              PALETTE USING integer array name (array index)

 Комментарии: attribute  представляет цвет в палитре.
              color - действительный цвет,заменяющий attribute в па
              литре.Получаемый на экране цвет зависит от установленного
              экранного режима (см. оператор SCREEN).
                Оператор PALETTE  изменяет  цвет  текста или графики на
              новый цвет.Это изменение оказывает влияние на любой  пос-
              ледующий текст или графику.Кроме этого,оператор позволяет
              задавать номера цветов,превышающих 15 из  числа имеющихся
              в  палитре.Можно отображать невидимый текст,а затем,изме-
              нив его цвет,сделать его видимым.Если в операторе PALETTE
              не указаны параметры,то применяется цвет умолчания.
                Оператор PALETTE USING заменяет все цвета палитры в од-
              ном  операторе.Integer array name присваивает имя массиву
              (матрице).Определяя палитру,используйте array index.
                Удостоверьтесь,что размеры целочисленного массива доста-
              точно велики,чтобы вместить всю  шестнадцатицветовую  па-
              литру с array index.
