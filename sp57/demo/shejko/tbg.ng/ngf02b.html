      Числа с удвоенной точностью соотносятся с числами одинарной  точ-
 ности  также,как  расширенные целые числа с целыми.Они занимают 8 байт
 на число,имеют большой ряд (10^308) и большую точность (16 цифр  про-
 тив 6 при одинарной точности),а время их обработки самое большое.
      Примечание:Turbo-Basic использует для арифметических  операций  с
 плавающей  запятой  стандарт IEEE,а не собственный стандарт MICROSOFT,
 используемый интерпретатором Basic.Поэтому Вы должны использовать спе-
 циальные функции трансляции MICROSOFT/IEEE (CVMS,CVMD,MKMS$,MKMD$) для
 считывания данных с плавающей запятой из файлов произвольного доступа,
 созданных интерпретирующим Basic.Если Вы не знаете ,какой тип с плава-
 ющей запятой использовать,мы советуем удвоенную точность,но это  зави-
 сит от Вашего конкретного случая.
