Синтаксис:$EVENT{ON│OFF}

Комментарии:Если Ваша программа содержит прерывания  какого-либо вида
              (например,ON  KEY,ON COM),то $EVENT ON разрешает проверку
              наличия прерывания в перерывах между  выполнением команд,
              в  противном  случае это проверка запрещается.$EVENT даст
              Вам возможность вкл/выкл проверку на прерывание  в нужных
              местах программы.

Пример:     ON TIMER(1) GOSUB Time
                TIMER ON
                PRINT "Slow loop"
                x=timer
                FOR i=1 TO 10000
                  i=i+i-i
                NEXT
                y=timer
                PRINT "loop time is"y-x
                $EVENT OFF
                x=timer
                PRINT "Fast loop"
                FOR i=1 TO 10000
                  i=i+i-i
                NEXT
                y=timer
                PRINT "loop time is"y-x
                END

                Time:
                  FOR j=1 TO 1000:j=j+j-j:NEXT j:PRINT
                RETURN
