 Прописные буквы  описывают  части команд,которые должны записываться в
 программе дословно.
 Квадратные скобки (  [] ) означают,что заключенная в них информация не-
 обязательна. Фигурные скобки (  {} ) указывают на два или более вариан-
 тов,один из которых должен быть выбран.
 Элемент ( ... ) указывает,что часть команды  может  повторяться  столько
 раз,сколько это необходимо. Наличие в строке программы одной точки оз-
 начает опускание одной или более строк текста.
