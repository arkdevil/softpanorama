 Комментарии: count - целочисленное выражение от 0 до 32767,указывающее
              на  количество  байт,которое надо считать в строку string
              variable из файла с номером filenum. Данные считываются с
              текущей  позиции  файла  (которая может устанавливаться с
              помощью SEEK).После считывания текущая позиция  перемеща-
              ется на указанное количество байт. GET$,PUT$ и SEEK обес-
              печивают самый нижний уровень в обработке файлов.
See Also: OPEN PUT$ SEEK
