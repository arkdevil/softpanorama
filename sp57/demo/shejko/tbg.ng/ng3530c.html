 Синтаксис:   LPRINT [список выражений [;]]
              LPRINT USING format strind;expression list

 Комментарии: expression  list  - ряды разделяемых запятой или точкой с
              запятой  численных  и/или   строковых   выражений.'Format
              strind' (строка формата) содержит информацию о форматиро-
              вании.
              LPRINT USING выполняет такие же операции,  что и PRINT USING,
              за  исключением  того,что  пересылают данные на печать,а не на
              экран.

 Примечание:  распечатав 80 знаков на строке,Turbo-Basic делает по
              умолчанию перевод строки.Эту характеристику можно изме-
              нить с помощью оператора 'WIDTH'.
