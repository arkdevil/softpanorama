 Синтаксис:   ON TIMER (n) GOSUB label

 Комментарии: label идентифицирует первый оператор подпрограммы;n - це-
              лочисленное выражение,определяющее время  ожидания  между
              прерываниями (в сек.) от 1 до 86400 (24 часа).
               Оператор ON TIMER должен быть разрешен  оператором TIMER
              ON.
               Оператор TIMER OFF отменяет действие ON TIMER и  сбрасы-
              вает счетчик секунд на 0.
               Вслед за прерыванием программы по  сигналу  от  счетчика
              выполняется  неявный  оператор TIMER STOP,предупреждающий
              повторный вызов подпрограммы.Заключительное RETURN  авто-
              матически  выполняет  TIMER ON,если только в программе не
              было явного TIMER OFF.
               В целях более точного управления версией Turbo-Basic ис-
              пользуют метаоператор $EVENT.

See Also: $EVENT TIMER
