 Синтаксис:   1-я форма        color=POINT(x,y)
              2-я форма        y=POINT(option)

 Комментарии: (x,y) определяет координаты графического элемента,цвет
              которого будет возвращен.Если x или y являются некор-
              ректными величинами,то POINT возвращает -1.
              option - числовое выражение,управляющее работой POINT:
                                                      --------------------------------------------------------
              Значение                       Эффект
              --------------------------------------------------------
              0           Возвращает текущую физическую координату x.
              1           Возвращает текущую физическую координату y.
              2           Возвращает текущую мировую координату x.
              3           Возвращает текущую мировую координату y.
              --------------------------------------------------------
              1-я форма POINT часто используется для точечной
              мультипликации.
