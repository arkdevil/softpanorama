 Синтаксис:   установка системной даты     DATE$=s$
              чтение системной даты        s$=DATE$

 Комментарии: присвоение DATE$ строковой переменной устанавливает  сис-
              темную дату.Присвоение строковой переменной значения DATE
              $ возвращает системную дату в виде 10 знаковой  строки  в
              формате  mm-dd-yyyy,где mm - месяц,dd - день,  yyyy - год
              (в пределах от 1980 до 2099). Для изменения системной да-
              ты  Ваша строка должна содержать данные в одном из форма-
              тов:

              mm-dd-yy
              mm/dd/yy
              mm-dd-yy
              mm/dd/yyyy

              Например,

              DATE$="10-25-86"

              устанавливает дату на 25 октября 1986г.

 Пример:      PRINT "The system date is ";DATE$
              INPUT "Please enter the new date in the form MM-DD-YY";_
                    NewDate$
              DATE$=NewDate$
              PRINT "The system date is ";DATE$
              END
