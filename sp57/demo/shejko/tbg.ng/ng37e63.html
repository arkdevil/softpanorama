 Синтаксис:   ON n GOSUB label [,label]...

 Комментарии: n - численное выражение в пределах от  0  до  255.  label
              идентифицирует  оператор  ветвления.  n  определяет,какой
              метке  label  передается  управление.
              Выполнение каждой программы должно завершаться оператором
              RETURN с тем,чтобы возвратиться к следующему  за ON/GOSUB
              оператору.  Блочные  операторы SELECT и IF тоже могут вы-
              полнять многосторонние ветвления,но являются более гибки-
              ми.
