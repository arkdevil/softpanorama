 Эта команда состоит из ряда опций:
      Colors (цвет) - с помощью этого параметра Вы можете менять цвет любого
 параметра меню Turbo-Basic:окна,окантовки,текстовой информации.
      Directories (справочник) - определяет накопитель и путь для пара-
 метров Include,Executable,Turbo.
 Для параметра  Include (применяется при компилировании файлов с дирек-
 тивой INCLUDE) можно задать несколько накопителей и/или путей,для двух
 других - только один.  Miscellaneous (смешанный тип) - состоит из двух
 параметров:
      Auto save edit - при включенном состоянии автоматически сохраняет
 рабочую программу перед ее прогоном.
      Backup source  files - применяется для автоматического резервиро-
 вания Ваших файлов.Переключите эту опцию в положение выключено,еслт Вы
 не  хотите создания резервных файлов при операции хранения.  Все опции
 компилятора и другие установочные величины хранятся в файле конфигура-
 ции.Это  позволит Вам создать один раз нужную операционную среду (нап-
 ример,включить при отладке опции компилятора,расширить  окно  Trase  и
 включить  флажок  трассировки),записать ее параметры на диск,  а затем
 загружать их с диска.Это даст Вам существенную  экономию  времени  при
 настройке среды Turbo-Basic.Следующие две опции позволят Вам загружать
 и сохранять файлы конфигурации:
      Load Options/Window/Setup  -  загружает с диска файл конфигурации
 Turbo-Basic.
      Save Options/Window/Setup  - записывает на диск файл конфигурации
 (имя умолчания TBCONFIG.TB).При желании Вы можете  сохранить параметры
 конфигурации в TB.EXE.
