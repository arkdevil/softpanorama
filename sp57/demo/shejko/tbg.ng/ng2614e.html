 Синтаксис:   s$=COMMAND$

 Комментарии: COMMAND$  возвращает  строку,набранную по подсказке DOS в
              процессе вызова  программы,за  исключением  самого  имени
              программы  (в  некоторых  учебниках этот текст называется
              'трейлер').Используется для анализа опций или ключей, ис-
              пользуемых  при  вызове  программы.  При прогоне програм-
              мы,использующей COMMAND$,из  TurboBasic,используйте опцию
              меню Option для ввода строки, возвращаемой COMMAND$.

 Ограничения: максимальная длина строки из DOS 127 знаков.

 Пример:      PRINT "The command line parameters passed to this prog_
              ram are"
              PRINT COMMAND$

 Скомпилировать эту программу и прогнать ее дважды,изменяя параметер в
 меню опций.
