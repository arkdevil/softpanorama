 Синтаксис:   PRINT #filenum,[USING format string;]expression list[;]

 Комментарии: filenum - номер открытого файла.
              format string - необязательная последовательность  форма-
              тируемых   знаков  (описаны  в  операторе  PRINT  USING).
              expression list - последовательность числовых и/или стро-
              ковых выражений,выводимых в файл.
               PRINT # пересылает информацию  в  файл  таким  же  обра-
              зом,как PRINT пересылает ее на экран.
               Т.к. оператор INPUT # требует разделения значений  запя-
              той,то в операторе PRINT # надо разделять данные:

              PRINT #1,1","2","3

              При этом в файле будет записано:

              1 , 2 , 3

              Оператор WRITE # автоматически разделяет данные,выводи-
              мые в файл,запятыми.
