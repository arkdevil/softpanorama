 Синтаксис:   KILL filespec

 Комментарии: filespes идентифицирует имя удаляемого файла и может  со-
              держать  путь  к файлу и обобщающие знаки (?,*).
              KILL не может удалить подкаталог.Для этой цели существует
              оператор RMDIR.

 Пример:      ON ERROR GOTO FileError
              INPUT "Please enter the file to delete: ",FileName$
              IF FileName$><"" THEN
                KILL FileName$
              END IF
              END

              FileError:
                PRINT "Error ";ERR
                END
