 Синтаксис:   IF integer expression [.] THEN
              .
              . statement(s)
              .
              [ELSEIF integer expression [.] THEN
              .
              . statement(s)
              [ELSE
              .
              . statement(s)]
              .
              END IF

 Комментарии: при  выполнении блока операторов IF в первую очередь про-
              веряется истинность выражения в операторе IF.Если  значе-
              ние  FALSE  (нулевое),то выполняются по порядку операторы
              ELSFIF (их количество неограниченно),если  TRUE  (ненуле-
              вое),то  выполняются операторы за словом THEN,а затем уп-
              равление передается на следующий за END IF  оператор.Опе-
              ратор(ы)  после  необязательного ELSE выполняются,если ни
              один из прежних тестов не был TRUE.Операторы блока IF мо-
              гут  вкладываться друг в друга,т.е.  любой оператор после
              THEN может содержать блок IF. Блок IF должен оканчиваться
              оператором END IF.
