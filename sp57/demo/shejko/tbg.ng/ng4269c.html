 Копирование примеров из данного пособия в
 вашу программу.

   При нажатии комбинации клавиш Alt-W на экране
   появляется курсор. Для того что-бы отметить
   видимую часть примера сначала надо подвести
   курсор (клавишами управления) к началу выделяемого
   фрагмента нажать клавишу ENTER, подвести к концу
   фрагмента нажать клавишу ENTER. После этого
   выделенный фрагмент записывается на диск. Для
   этого надо нажать F и ответить на запрос
   имени файла для записи.

   Записаный в файл пример можно удобно вставлять
   в свою программу командой Турбо-Бейсика Ctrl-K-R.
