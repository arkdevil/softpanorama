 Синтаксис:   LINE INPUT [;][prompt string:]string variable

 Комментарии: prompt  string  -  строка  подсказки,выводимая  на экран.
              string  variable  -  строковая  переменная,которой  будет
              присвоено  введенное  значение.
              LINE INPUT применяется вместо INPUT,если строка ввода со-
              держит  ограничители (запятые),которые нарушили бы работу
              TNPUT.Если за оператором LINE INPUT стоит точка  с  запя-
              той,то после ввода строки курсор останется на строке вво-
              да.

 Пример:  PRINT "Enter several fields of input,you needn't "
              LINE INPUT "worry about delimiting then in any way:"_
              DummyStr$
              PRINT DummyStr$
              END
