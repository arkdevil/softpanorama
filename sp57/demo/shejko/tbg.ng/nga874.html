      Выполняет программу,находящуюся  в  памяти.Если  скомпилированная
 программа  уже находится в памяти и источник этой программы не был из-
 менен в редакторе с момента последней команды Compile или Run и ни од-
 на  опция не была изменена,то программа выполняется немедленно.  В про-
 тивном случае исходная программа сначала  компилируется,а  лишь  затем
 выполняется.   Выполняемую   программу   можно   прервать  комбинацией
 Ctrl-Break,если была установлена опция прерывания.  Окно Message отоб-
 ражает результаты компиляции программы,а окно Run - ее выполнения. Для
 увеличения масштаба  окна  Run  во  время  прогона  программы  нажмите
 Alt-F5.
