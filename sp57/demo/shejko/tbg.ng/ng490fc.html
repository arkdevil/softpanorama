  Синтаксис: y=SQR (числовое выражение)

  Замечания: числовое выражение должно быть больше или равно нулю.
        SQR вычисляет квадратные корни с использованием бо-
        лее быстрого алгоритма, чем метод вычисления квадратного  корня  пос-
        редством возведения в степень 0,5 (т.е. Y=SQR (x) требует на исполне-
        ние меньше времени, чем y=x.5).
              Попытка взять квадратный корень отрицательного числа приводит к
        ошибке  5  периода  прогона, вызов незаконной функции. SQR возвращает
        результат двойной точности.
  Пример:              Таблица квадратных корней
                       'отметить две колонки
                       PRINT "ЧИСЛО" TAB(10) "ЗНАЧЕНИЕ КВАДРАТНОГО КОРНЯ"
                       PRINT
                       FOR N = 1 TO 10
                      'использовать TAB для вывода квадратного корня  в  пра-
        уильной колонке, использовать SQR для получения квадратного корня
                      PRINT N, TAB(15) SQR(N)
                      NEXT N
                      END
