      Команда поиска  инициализируется Ctrl-QA с последующим вводом це-
 почки искомых символов и символов замены.Затем Вы можете выбрать вари-
 анты  выполнения этой команды:G - для глобального поиска;N - для авто-
 матической (без подсказки) замены старых символов на  новые. Закончите
 ввод опций клавишей Пуск.
      Для поиска цепочки символов без их последующей замены введите  ко-
 манду Ctrl-QF.Для повторения поиска нажмите Ctrl-L
