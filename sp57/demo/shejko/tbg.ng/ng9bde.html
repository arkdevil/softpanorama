      Эта команда предоставляет Вам меню с несколькими опциями:
      Load (загрузка) - позволяет Вам загрузить рабочий  файл,  который
 затем можно редактировать,компилировать,выполнить или сохранить. После
 нажатия L Вам дается подсказка имени файла.Вы можете ввести:
 1. Любоем корректное  имя файла (по умолчанию расширение файла .BAS).
 2. Выбрать имя файла из справочника.Для  этого  сразу  нажмите  'Пуск'
 (если  необходимо,укажите перед этим дисковод и/или маску).После этого
 будет выведен каталог дисковода,где с помощью светового курсора Вы мо-
 жете выбрать нужный Вам файл.
      После загрузки файла Turbo-Basic автоматически переходит в редак-
 тор.Если Вы сохраните рабочий файл на диске,то старая версия программы
 будет переименована в файл с расширением .BAK.
      Если при загрузке Turbo-Basic не находит файл с указанным именем,
 то он создает новый файл.
      New (новая) - эта опция стирает текст,находящийся в данный момент
 в редакторе и дает файлу сохранения имя NONAME.BAS (имя умолчания).
      Save (хранение) - сохраняет текущий рабочий файл на диске. Старой
 версии программы,если она есть,придается  расширение  .BAK.  Если  имя
 файла NONAME.BAS,то Вам будет дана попытка переименовать его до записи
 на диск.
      Write to  (запись)  -  используется  для записи рабочего файла на
 диск под новым именем.
      Main file  (основной файл) - предоставляет возможность при работе
 с большими программами,когда возможна ситуация превышения знаковой ем-
 кости редактора,разбивать эти программы на независимые части.Эти части
 хранятся на диске и редактируются как отдельные файлы,но когда прихо-
 дит  время компиляции,они сводятся вместе в одну программу.Совместно с
 метаоператорами $INCLUDE выбор основного файла позволит Вам компилиро-
 вать программы,состоящие из многочисленных частей. Исходный модуль,ко-
 торый  Вы  определяете  как  основной  файл,должен   иметь   директивы
 $INCLUDE.Как  всегда,для  редактирования любого модуля программы (даже
 основного модуля),сделайте его текущим загружаемым  файлом.По  команде
 Compile Turbo-Basic проверяет,отличается ли от основного файла текущий
 рабочий файл.Если нет,то рабочий файл компилируется.В противном случае
 Вам  дается  подсказка  для сохранения рабочего файла и затем основной
 файл загружается в память.Если во время  компиляции  обнаружена  ошиб-
 ка,то файл с ошибкой автоматически загружается в память для редактиро-
 вания.После корректировки Вам дается возможность сохранить этот файл,а
 затем вновь загружается основной файл.
      Directory (справочник) - выводит на экран  каталог  дисковода  (с
 возможностью задания маски).
      Change dir (изменение справочника) - применяется для выбора  дру-
 гого каталога или другого дисковода.
      OS shell (оболочка) - возвращается в DOS,оставляя Turbo-Basic ре-
 зидентным.Управление  возвращается  Turbo-Basic с помощью команды Exit
 DOS.
      Quit (прерывание) - возвращает Вас в DOS (аналогична команде Alt-
 X).
