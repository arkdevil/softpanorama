 Синтаксис:   y%=CVI(2-byte string)
              y&=CVL(4-byte string)
              y!=CVS(4-byte string)
              y#=CVD(8-byte string)

 Комментарии: CVI преобразует 2-байтовую цепочку в целое число.
              CVL преобразует 4-байтовую цепочку в расширенное целое
              число.
              CVS преобразует 4-байтовую цепочку в значение с одинар-
              ной точностью.
              CVD преобразует 8-байтовую цепочку в значение с удвоен-
              ной точностью.
              Эти функции предназначаются для работы с файлами произ-
              вольного доступа,не путайте их с функцией VAL.

 Пример:      OPEN "R",#1,"CV.DTA",18
              FIELD #1,2 AS Int$,4 AS LongInt$,4 AS SinglePre$,_
                       8 AS DoublePre$
              MaxInt%=32767

              FOR i%=1 TO 5
                LSET Int$=MKI$(i%)
                LSET LongItn$=MKL$(i%+CLNG(MaxInt%))
                LSET SinglePre$=MKS$((CSNG(i%*CLNG(MaxInt%)))
                LSET DoublePre$=MKD$(MaxInt% i%)
                PUT #1,i%
              NEXT i%

              FOR i%=1 TO 5
                GET #1,i%
                PRINT CVI(Int$),CVL(LongInt$),CVS(SinglePre$),_
                      CVD(DoublePre$)
              NEXT i%

              CLOSE #1
              END
See Also: MKD$ MKI$ MKL$ KKS$
