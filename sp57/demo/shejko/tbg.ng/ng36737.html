 Синтаксис:  s$=MKS$(выражение с одинарной точностью)

 Комментарии: этa  функция является частью процесса получения численных
              значений в файлах с произвольной выборкой.Поскольку  опе-
              раторы,записывающие информацию в буфер файла с произволь-
              ной выборкой (LSET,RSET),управляют  только  строками,чис-
              ленные   данные  следует  преобразовать  в  строковые  до
              пересылки (PUT) ф файл.
              MKS$ возвращает 4 -байтовый строковый эквивалент значения
              с одинарной точностью.А  MKD$ возвращает 8-байтовый строко-
              вый эквивалент значения с двойной  точностью. Дополняющие
              функции CVI,CVL,CVS,CVD предназначены для использования в
              процессе считывания содержимого файла с произвольной  вы-
              боркой.
