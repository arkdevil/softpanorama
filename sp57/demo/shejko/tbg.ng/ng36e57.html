 Синтаксис:  s$=MKMD$(выражение с двойной точностью)

 Комментарии: эти  функции предусмотрены только для совместимости с су-
              ществующими файлами с произвольной выборкой,в которых со-
              держатся  числа  с плавающей запятой в Microsoft-формате.
              MKMS$ формирует 4-байтовую строку с  одинарной точностью;
              MKMD$ формирует 8-байтовую строку с двойной точностью.
