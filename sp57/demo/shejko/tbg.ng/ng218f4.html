Синтаксис:   y=ASC(строковое выражение)

Комнтарии: ASC  возвращает  код  ASCII  (от  0 до 255) первого знака
              строкового выражения.Обратное действие  по преобразованию
              кода  в символ выполняет функция CHR$.  Строковое выраже-
              ние,посылаемое на ASC,не должно быть пустой  строкой (при
              этом формируется ошибка 5:вызов несанкционированной функ-
              ции).

 Пример:       PRINT "The ASCII value of A is ";ASC("A")
See Also: CHR$
