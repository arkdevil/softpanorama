 Синтаксис:   CLEAR

 Комментарии: CLEAR является  безпараметной  командой  и  устанавливает
              числовые и строковые переменные на 0,очищает массивы.

 Примечание:  используйте ERASE для выборочного обнуления массивов.

 Ограничения: CLEAR не устанавливает сегменты и стек.

 Различия: интерпретирующий Basic динамически  устанавливает оператором
              CLEAR  пространство стека и данных,в Turbo-Basic это про-
              исходит во время компиляции.

 Пример:      DIM Array$(1)
              PLAY "MB L1 EFGABCDEF"
              ON PEN GOSUB PenHandler
              ON STRIG(0) GOSUB ButtonOHandler

              Array$(0)="This is a string in Array$(0)"
              Array$(1)="This is a string in Array$(1)"

              PRINT USING "Array element 1: &";Array$(0)
              PRINT USING "Array element 2: &";Array$(1)
              PRINT
              INPUT "Press <ENTER> to execute CLEAR statement";Dimmy$

              CLEAR

              PRINT
              PRINT USING "Array element 1: &";Array$(0)
              PRINT USING "Array element 2: &";Array$(1)
              PRINT
              PRINT "The string variables are now empty."
              END

              PenHandler:
              RETURN

              ButtonOHandler:
              RETURN
