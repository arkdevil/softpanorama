 Синтаксис:   LINE INPUT #filenum,string variable

 Комментарии: filenum - номер файла,строка из  которого  считывается  в
              строковую переменную string variable с игнорированием ог-
              раничительных запятых в тексте. Желательно,чтобы файл был
              сформирован с помощью WRITE#.

 Пример:      SUB MakeFile
                OPEN "LINEINP.DTA" FOR OUTPUT AS #1
                StringVariable$="There's trouble in River City."
                Integer%=1000
                FloatingPoint!=30000.1234
                WRITE #1,StringVariable$,Integer%,FloatingPoint!
                CLOSE #1
              END SUB

              SUB ReadFile
                OPEN "LINEINP.DTA" FOR OUTPUT AS #1
                StringVariable$=""
                LINE INPUT #1,StringVariable$
                PRINT StringVariable$
              END SUB

              CALL MakeFile
              CALL ReadFile
              END
