  Синтаксис:   CLOSE [[#]filenum[,[#]filenum]...]

 Комментарии: filenum - номер файла,который надо закрыть.  CLOSE преры-
              вает  соответствие  между номером файла и файлом,установ-
              ленное оператором OPEN,и закрывает файл.  CLOSE без аргу-
              ментов закрывает все файлы и устройства (так же поступают
              RESET,END,STOP,RUN).

 Пример:      OPEN "LPT1:" as #1
              PRINT #1,"This is a test"
              CLOSE #1

              OPEN "CLOSEFIL.ONE" AS #1
              PRINT #1,"This is a test"
              CLOSE #1

              END
See Also: END OPEN STOP
