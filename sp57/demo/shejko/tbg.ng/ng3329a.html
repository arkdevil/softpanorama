 Синтаксис:   s$=LEFT$(string expression,n)

 Комментарии: n  - целочисленное выражение в пределах от 0 до 32767,оп-
              ределяющее количество знаков,которое должно быть  возвра-
              щено.Если n больше длины string expression,то возвращает-
              ся вся строка,если n равно 0,то нулевая.

 Пример:      PRINT LEFT$("Hello out",3)
See Also: MID$ RIGTH$
