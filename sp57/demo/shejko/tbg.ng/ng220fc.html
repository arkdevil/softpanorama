 Синтаксис:   BLOAD filespec[,address]

 Комментарии: filespec определяет имя загружаемого файла. address явля-
              ется необязательным числовым выражением от 0 до  65535  и
              определяет  смещение  в текущем сегменте,с которого будет
              загружен файл.Если address опущен,то файл  размещается  с
              адреса,с  которого  он  был  записан  на  диск по команде
              BSAVE.  Т.к. никакого контроля за адресом файла не произ-
              водится, то можно нечаянно испортить системную информацию
              или свою программу,что приведет к аварии системы.

 Ограничения: BLOAD не может напрямую загружать память EGA (расширен-
              ного графического адаптера).

 Пример:      DEF SEG=&HB800
             BLOAD "Picture",0
See Also: BSAVE DEF SEG
