 Синтаксис:   LOCATE [row][,[column][,[cursor][,start][,stop]]]

 Комментарии: row - целочисленное выражение,определяющее строки экрана,
              на  которую должен быть установлен курсор (1-25).
              column аналогичным образом определяет колонку курсора (1-
              80).
              cursor - числовое значение,определяющее видимость  курсо-
              ра:1 - видимый,0 - невидимый.  start и stop - целочислен-
              ные выражения,определяющие размер курсора,т.е. количество
              строк  развертки,которые будут формировать курсор.Верхняя
              строка развертки имеет значение 0,нижняя - 7 (для цветных
              адаптеров) или 13 (для монохромных адаптеров).

 Пример:      CLS
              INPUT "Input X and Y coordinates: ",x%,y%
              LOCATE x%,y%,1,4,5
              PRINT "Hi"
              END
