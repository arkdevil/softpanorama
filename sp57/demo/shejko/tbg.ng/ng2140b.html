Синтаксис:   $STATIC

Комментарии: $STATIC не имеет аргументов и объявляет тип  массивов  по
              умолчанию  статическим.Пространство для этих массивов ре-
              зервируется во  время  компиляции.Массивы  с  постоянными
              размерами всегда статические,если только они не объявлены
              локальными внутри функции или процедуры.Оператор ERASE не
              удаляет  из памяти этот тип массивов,а лишь обнуляет его.
              Заметьте,что  метаоператор  $STATIC,несмотря  на  внешную
              схожесть,не имеет ничего общего с оператором STATIC,кото-
              рый объявляет о специальном типе локальной  переменной  в
              рамках  процедуры  или  функции.  Умолчание компилятора -
              $STATIC.

Пример:      $STATIC
              PRINT "Memory available: ",FRE(-1)
              DIM A(50)
              PRINT "Memory available: ",FRE(-1)
              n=50
              DIM B(n)
              PRINT "Memory available: ",FRE(-1)
              ERASE A
              PRINT "Memory available: ",FRE(-1)
              ERASE B
              PRINT "Memory available: ",FRE(-1)
              END
See Also: DIM $DYNAMIC ERASE FRE
