 Синтаксис:   y=PMAP(x,option)
 Комментарии: x -координата отображаемой точки.
              option - целочисленное выражение от 0 до 3,управляющее
              функцией PMAP:
              --------------------------------------------------------
              Значение                        Эффект
              --------------------------------------------------------
              0              Установить соответствие мировой коорди-
                             наты x физической координате x.
              1              Установить соответствие мировой коорди-
                             наты y физической координате y.
              2              Установить соответствие физической коор-
                             динаты x мировой координате x.
              3              Установить соответствие физической коор-
                             динаты y мировой координате y.
             --------------------------------------------------------
              PMAP выполняет масштабирование,определяемое самым послед-
              ним оператором WINDOW без вывода информации на экран.
