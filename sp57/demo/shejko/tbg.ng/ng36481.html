 Синтаксис:  s$=MKL$(длинное целочисленное выражение)

 Комментарии: эта  функция является частью процесса получения численных
              значений в файлах с произвольной выборкой.Поскольку  опе-
              раторы,записывающие информацию в буфер файла с произволь-
              ной выборкой (LSET,RSET),управляют  только  строками,чис-
              ленные   данные  следует  преобразовать  в  строковые  до
              пересылки (PUT) в файл.
              MKL$ возвращает 4-байтовый строковый эквивалент длинного цело-
              го  числа.
              Дополняющие функции CVI,CVL,CVS,CVD предназначены для
              использования в процессе считывания содержимого файла с
              произвольной  выборкой.
