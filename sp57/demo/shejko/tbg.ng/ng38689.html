 Синтаксис:   ON PER GOSUB label

 Комментарии: label идентифицирует первый оператор в подпрограмме обра-
              ботки светового пера.
              Оператор ON PEN должен быть разрешен оператором  PEN  ON.
              После  выполнения оператора PEN ON перед выполнением каж-
              дого следующего оператор проверяется активизация светово-
              го пера.При активизации светового пера вызывается указан-
              ная подпрограмма.  Оператор PEN OFF отменяет контроль  за
              световым  пером.
              Вслед за прерыванием светового пера  выполняется  неявный
              оператор  PEN STOP ,предупреждающий повторный вызов подп-
              рограммы до ее завершения.Заключительное RETURN  подпрог-
              раммы  обработки  автоматически  исполняет  оператор  PEN
              ON,если только явно не исполнялся оператор PEN OFF. В це-
              лях  более точного управления версией Turbo-Basic исполь-
              зуют метаоператор $EVENT.
