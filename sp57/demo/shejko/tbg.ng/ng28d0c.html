 Синтаксис:   DECR variable[,size]

 Комментарии: variable  - числовая переменная,значение которой уменьша-
              ется  на  величину  необязательного  числового  выражения
              size.По  умолчанию size равно 1.
              Для увеличения значения переменной используется INCR.

 Пример:      i%=15
              j%=500

              WHILE i%>0
                PRINT i%,j%
                DECR i%
                DECR j%,5
              WEND
              END
See Also: INCR
