 Синтаксис:   LINE [(x1,y1)]-(x2,y2)[,[color][,B[f]][,pattern]]

 Комментарии: (x1,y1) и (x2,y2) являются  координатами  двух  точек  на
              графическом экране либо в абсолютной,либо в относительной
              форме.
              color - целочисленное выражение,описывающее цвет вычерчи-
              ваемой линии или цвет заполнения  прямоугольника.
              pattern -  целочисленная маска,налогаемая на рисунок. Для
              вычерчивания линии до точки (x,y) из начальной  точки LPR
              в цвете умолчания задайте:

              LINE -(x,y)

              Для изменения цвета линии введите параметер color:

              LINE -(x,y),2

              Для вычерчивания линии из точки (x1,y1) в точку (x2,y2):

              LINE (x1,y1)-(x2,y2)

              Для вычерчивания  прямоугольника  с  верхним  левым углом
              (x1,y1) и нижним правым углом (x2,y2):

              LINE (x1,y1)-(x2,y2),,B

              Для заполнения прямоугольника цветом:

              LINE (x1,y1)-(x2,y2),2,BF

              Для вычерчивания несплошной линии  применяется  параметер
              pattern;этот  параметер  не  влияет на заполнение прямоу-
              гольника:

              LINE (x1,y1)-(x2,y2),,,maskword

Пример:      SCREEN 1,0
              LINE (0,0)-(319,199)
              LINE (0,100)-(319,100),,,&HCCCC
              LINE (10,20)-(60,70),2,BF
              READ x,y
              PSET (x,y)
              FOR n=1 TO 11
                READ x,y
                LINE -(x,y)
              NEXT n
              LOCATE 9,9:PRINT "ORLAND"
              END
              DATA 10,20,50,20,55,25,55,40,50,45,10,45
              DATA 50,45,55,50,55,65,50,70,10,70,10,20
