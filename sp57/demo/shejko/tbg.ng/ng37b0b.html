 Синтаксис:   ON ERROR GOTO label

 Комментарии: label идентифицирует первую  строку  программы  обработки
              ошибок.Чтобы  возвратиться в основную программу после об-
              работки ошибки,используйте оператор  RESUME  (продолжить)
              Для  запрещения прерываний при обнаружении ошибок исполь-
              зуйте оператор ON ERROR GOTO 0.  Если программу  .EXE  из
              DOS прогоняют без прерывания по ошибкам,то любая возника-
              ющая ошибка приведет к распечатке сообщения  и завершению
              выполнения программы.Сообщение выводится в следующем фор-
              мате.
                    Error errnum  at pgm-str=address Если ошибка возни-
              кает при прогоне программы в TurboBasic,редактор  показы-
              вает ошибочное место.Краткое описание ошибки будет дано в
              строке состояния:
                    Error errnum:error message
