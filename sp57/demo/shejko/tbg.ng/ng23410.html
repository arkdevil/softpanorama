 Синтаксис:   CALL INTERRUP

 Комментарии:  - целочисленное выражение от 0 до 255,определяющее  но-
              мер  прерывания.Во время обработки прерывания регистровый
              буфер содержит текущую информацию о регистрах процессора.
              Технический  справочник  DOS содержит полную информацию о
              функциях,доступных  благодаря  механизму  CALL  INTERRUP.
              Программист несет ответственность за сохранение регистров
              SS,SP.

 Ограничения: перед оператором CALL INTERRUP должны инициализироваться
              соответствующие регистры ЦПУ с помощью оператора REG.

 Пример:      $INCLUDE "REGNAMES.INC"
              REG %AX,&H0200                       'AH=&H02
              REG %DX,REG(%DX) AND &HFF02          'DL=&H02

              CALL INTERRUPT &H21
See Also: CALL ABSOLUTE REG
