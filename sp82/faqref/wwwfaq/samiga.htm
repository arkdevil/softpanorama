Amiga Servers

AWS
AWS is the first server written specifically for the Amiga. Documentation is available from <URL:http://www.phone.net/aws/> , and the distribution can be downloaded by anonymous FTP from <URL:ftp://max.physics.sunysb.edu/pub/amiga/amosaic/> .
NCSA
NCSA's Unix server has been ported to the Amiga, and is bundled with the AMosaic browser; however, a web page about the port is no longer available.
