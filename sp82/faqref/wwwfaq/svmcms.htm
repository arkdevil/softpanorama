VM/CMS Servers

Webshare
A VM/CMS web server is available from Beyond Software Incorporated. It was written entirely in REXX and supports CGI programming in that language as well as more traditional web server features. Imagemaps are limited to rectangular regions. <URL:http://www.beyond-software.com/Software/Webshare.html>
VM:Webserver
Sterling Software, Inc. also offers a VM/CMS based web server. CGI and other web server standards are supported. Web browser-based setup and maintenance features are included. <URL:http://www.vmd.sterling.com/general/products/viwindex.html>
