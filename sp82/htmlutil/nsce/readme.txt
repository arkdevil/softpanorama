INTRODUCTION AND SETUP INSTRUCTIONS

Easy offline-access to all Web pages that have been cached by
Netscape Navigator.
Contains fulltext search and the ability to extract entire
document trees.
Organizing functions like intelligent domain sorting, hiding and
deleting documents, helps you to keep track of important things
in the browser cache universe.
Netscape 2.0 or later (32bit) is required.

System requirements:  Win95/WinNT 4.0

INSTALLATION

This software is shipped as a self-extracting archive. The normal
file name is NSCE@.EXE (or NSCEG@.EXE for the German version).

To install this software:

- Create a new folder for this software.
- Copy the archive to this folder.
- Double-click the archive to decompress it. This will create several
  new files.
- Create a new Shortcut for the program NSCE.EXE within the Start Menu
  or on the Desktop.
- More information can be found in the online help.
