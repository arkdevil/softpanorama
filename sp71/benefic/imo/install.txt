
Установка Internet Mail Office v1.03
--------------------------------------

Процесс установки IMO запускается с консоли сервера командой:
	LOAD A:SETUP

Для работы SETUP необходима интерфейсная библиотека NWSNUT.NLM (входит в
поставку ОС Novell NetWare 3.12, 4.X) и при инсталяции на OC Novell NetWare 3.X
необходим модуль AFTER311.NLM.
Последовательность запуска для OC Novell NetWare 3.X:
	LOAD AFTER311
	LOAD NWSNUT
	LOAD A:SETUP

Последовательность запуска для OC Novell NetWare 4.X:
	LOAD NWSNUT
	LOAD A:SETUP

Главное меню инсталятора предлагает три альтернативы:
  - инсталлировать систему;
  - изменять конфигурацию устанновленной системы;
  - редактировать все конфигурационные файлы системы.

Инсталляция системы
-------------------
  Здесь SETUP пытается определить наличие почтовых систем - UUPC и
Novell MHS на томах сервера.
  После окончания сканирования высвечивается окно с указанием целевой
директории для инсталляции. Здесь можно просто нажать ENTER (либо
отредактировать при необходимости).

Конфигурация используемых шлюзов
--------------------------------
  Здесь можно выбрать установку шлюза для UUPC клиентов, шлюза для MHS клиентов,
либо оба шлюза одновремено. Также необходимо указать имя администратора
почты, путь к файлу псевдонимов (можно не указывать) и имя группы для
MHS шлюза. Для файла списка пользователей UUPC, директории почтовых ящиков
UUPC и MHS директории можно использовать значения по умолчанию.
  Следующая форма - конфигурация почтовых узлов.
Здесь необходимо указать для локального узла: имя узла, имя и пароль входа
в локальный узел (указывается только при работе системы в хост-режиме),
домен. Спул-директория имеет значение по умолчанию.
Для удаленного узла необходимо указать: имя узла, имя и пароль входа в
удаленный узел, список телефонов удаленного узла.
  Следующая форма - выбор и конфигурация модема.
Здесь необходимо выбрать подходящий модем из списка возможных и номер
коммуникационного порта, к которому подключен модем.
Затем можно изменить другие параметры модема.
  Тепер можно начинать процесс установки.

Для работы системы необходимо загрузить коммуникационный драйвер AIOCOMX и
библиотеку AIO, которые входит в состав ОС NetWare.
Например, для работы с портом COM1 необходимо задать:

	load AIO
	load AIOCOMX PORT=3F8 INT=4 NODE=0

Запуск системы IMO производится командой:

	load IMO

Данную последовательность команд можно занести в AUTOEXEC.NCF, что обеспечит
автоматическую загрузку системы при старте сервера.
