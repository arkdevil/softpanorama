        Определение Вашей версии Clarion Professional Developer
        ^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^
        Для этого необходимо запустить одну из программ  Clarion
(  CPRO, CRUN, CDES) c параметром "!" (без кавычек).  Номер вер-
сии будет выведен в виде: Version 2.1 (или 2.0)    Release  2XX[
X][ C], где XX[X] - номер релиза (batch, последняя  цифра необя-
зательна), а [C] - номер подрелиза (patch,  необязательно).  На-
пример Release 2108 - версия 2.1 релиз 8. 
        Получение нового релиза через BBS ClarioNOOS
        ^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^
        Внимание! Зарегистрированые пользователи Clarion Profes-
sional Developer могут получать новые подрелизы (patch)    через
нашу BBS. Patch - это изменение версии Clarion    СВОБОДНО расп-
ространяемое между изменениями релиза. Перед  использованием по-
дрелиза Вы должны иметь установленную на Вашем  компьютере ЗАРЕ-
ГИСТРИРОВАННУЮ версию Clarion Professional Developer     Release
2108.   После сообщения нам информации о Вас и  Вашего серийного
номера программного обеспечения Вы сможете  осуществить загрузку
файла с подрелизом с нашей BBS.

        Изменение релиза Вашего программного обеспечения
        осуществляется за минимальную плату указанную в списке
        цен через официального дилера Clarion Software.
