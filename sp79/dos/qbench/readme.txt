This file contains information on QBench, the DOS hard disk benchmark from 
Quantum Corporation which measures data access time and data transfer rate.

The complete set of files for QBench version 1.3 consists of:

     QBINSTR.COM  -  The QBench instructions.  Type QBINSTR to display.
     QBENCH.EXE   -  The QBench program.
     QBENCH.HLP   -  The help file used by QBENCH.EXE when help is invoked.
     QB130NEW.TXT -  A text file documenting QBench 1.3 enhancements.
     README.TXT   -  This file.

In order to understand and fully utilize QBench, all of these files must 
be available.  If you do not have all of these files, you may obtain an
up-to-date copy by calling the Quantum Product Support bulletin board.  
The bulletin board supports up to 9600 bps modems and uses 8 bits, no
parity, and 1 stop bit (8, N, 1).  The telephone number for the bulletin
board is

     408-894-3214
