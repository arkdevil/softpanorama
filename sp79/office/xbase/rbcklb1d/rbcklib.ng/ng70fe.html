 Clipper Summer '87
───────────────────────────────────────────────────────────────────────────────

 To build the Clipper Summer '87 demo program (RBckDemS.prg) :

 Compile with :

        Clipper RBckDemS

 Link with :

        PLink86 FILE RBckDemS LIB RBck_S87,Extend,Clipper

        - or -

        TLink RBckDemS,,NUL,RBck_S87+Extend+Clipper

        - or -

        Blinker FILE RBckDemS LIB RBck_S87,Extend,Clipper
