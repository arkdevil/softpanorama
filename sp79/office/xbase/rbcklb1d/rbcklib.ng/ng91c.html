 CA-Clipper 5.xx
───────────────────────────────────────────────────────────────────────────────

 To build the Clipper 5.xx demo program (RBckDemo.prg) :

 Compile with :

        Clipper RBckDemo /n

 Link with :
        For Clipper version 5.0x :

        RTLink   FILE RBckDemo LIB RBck_C50
        - or -
        Blinker  FILE RBckDemo LIB RBck_C50
        - or -
        ExoSpace FILE RBckDemo LIB RBck_C50,ExoSpace EXO PACK INT10

        For Clipper version 5.2x :

        RTLink   FILE RBckDemo LIB RBck_C52
        - or -
        Blinker  FILE RBckDemo LIB RBck_C52
        - or -
        ExoSpace FILE RBckDemo LIB RBck_C52,ExoSpace EXO PACK INT10
