 R_ClsBar() 
 Closes and clears an opened progress bar
──────────────────────────────────────────────────────────────────────────────


 Syntax
        R_ClsBar ( aBar )

 Arguments
        aBar            Opened progress bar (opened by R_OpnBar())

 Returns
        NIL

 Description
        Closes and clears an opened progress bar.
        The original screen (as it was before opening the bar) will be
        restored.

 Example
        See the example of R_OpnBar () and the demo program RBarDemo.PRG.
See Also: R_OpnBar() R_UpdBar()
