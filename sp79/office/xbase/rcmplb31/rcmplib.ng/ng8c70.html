 R_CmpStr() 
 Compresses a STRING using the Lempel-Ziv algorithm
──────────────────────────────────────────────────────────────────────────────

 Syntax
        cCmpStr = R_CmpStr ( cString )

 Arguments
        cString         String (or memo-field) to be compressed.

 Returns
        cCmpStr         Compressed string

                        If it isn't possible to compress the string or
                        if an error is detected while compressing,
                        the original (uncompressed) string will be
                        returned.

                        The errorcode can be obtained using the
                        R_LastErr() function. Possible errors :

                         0  No errors detected
                         1  Invalid parameter(s) passed
                        10  String couldn't be compressed
                        11  String was already compressed

 Description
        R_CmpStr() tries to compress the input string. This is done using
        the Lempel-Ziv algorithm.
        The compressed string can be decompressed using the R_DCmpStr()
        function.

 Example
        *-- Edit a compressed memo-field (Test->Txt)
        USE Test NEW

        REPLACE Test->Txt WITH ;
           R_CmpStr ( MemoEdit ( R_DCmpStr ( Test->Txt ) ) )

        More examples can be found in the demo program RCmpDemS.PRG.
