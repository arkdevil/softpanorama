 R_DCmpStr() 
 Decompresses a by R_CmpStr() compressed STRING
──────────────────────────────────────────────────────────────────────────────

 Syntax
        cString := R_DCmpStr ( cCmpStr )

 Arguments
        cCmpStr         A, by RCmpStr() compressed, string

 Returns
        cString         Original, decompressed string

                        If an error is detected while decompressing,
                        the compressed string will be returned.

                        The errorcode can be obtained using the
                        R_LastErr() function. Possible errors :

                        0 = CP_OKAY         No errors detected
                        1 = CP_INVALID_PARM Invalid parameter passed
                        3 = CP_NOT_RCMPLIB  String not compressed by R_CmpStr

                        See also the item ErrorCodes in this guide.

 Description
        R_DCmpStr() tries to decompress the (by R_CmpStr() compressed)
        input string.

 Example
        See R_CmpStr()
