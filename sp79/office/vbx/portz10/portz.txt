Introduction
------------
The PORTZ.VBX file is a custom control suitable for use with such programming
languages as Visual Basic, Visual C++ and Delphi. It gives access to the
computers I/O ports.

Compatibility
-------------
PORTZ.VBX is compatible with level 1 VBX controls.

Appearance
----------
When added to a program the control appears in the toolbox with the letters
'I/O' on it. It is non-sizable on a form and is invisible at run-time.

Properties
----------
In addition to the standard properties of Name, Index, Left and Top it also
has the following custom properties...
About (read only) - gives version information
Port - specifies the i/o port number (0-65535)
Data - specifies the data to write to or read from the specified port
Bits - specifies the i/o port size (8 or 16 bit)
Action (run-time only) - set to 1 to read from, or set to 2 to write to, the
specified port

The default Name property prefix and class name is 'Portz'.

Events
------
The control has no pre-defined events.
