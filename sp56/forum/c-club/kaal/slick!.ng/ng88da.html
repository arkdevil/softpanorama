   int far * far _videoparm(void);

   This function is used by low level drivers to get video parameters,
 when called first time it will fill up VIDEO structure with information
 which will be used for all subsequent calls until video mode changes or
 screen segment is set to 0.

 Parameters:

 None.

 Return:

 Pointer to physical/virtual screen.
See Also: VIDEO
