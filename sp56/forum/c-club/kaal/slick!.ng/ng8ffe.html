   void far _vputscp(int x,int y,char far *s,int color,int maxlen);

   Write ASCIIZ string to screen. Terminates if null byte is reached
 or maxlen characters have been written to screen. In case string was
 shorter than maxlen clear remaining length with spaces.

 Parameters :

 x,y        Coordinates for beginning.
 s          Pointer to ASCIIZ string.
 color      8 low bits are used as attribute for string.
 maxlen     maximum character count.

 Return :

 None.
See Also: _vputs() _vputsc()
