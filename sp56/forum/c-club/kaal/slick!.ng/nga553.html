    Some useful macros are defined in SLICK!.H to avoid function call
 overhead for very simple tasks. They're for cursor operation, and
 called :

  cursoroff()  Turn hardware cursor off
  cursoron()   Restore cursor size to the one used before opening the window
  restorcur()  Restore cursor position to where it was before opening window
  framesoff()  Release window frames (zoom window), doesn't affect screen
  DVbeginc()   Stop DesQview task switching
  DVendc()     Enable DesQview task switching
  DVpause()    Give up timeslice for another tasks
