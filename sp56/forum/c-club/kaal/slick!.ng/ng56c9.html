   int far pushwnd(void);

   This function will "push" window pointed down one level. See popwnd()
 for more information how to use it.

 Parameters :

 None.

 Return :

 1 if OK, 0 if there's only one active window or no active windows at all.

 Example :

 Look popwnd() example.
