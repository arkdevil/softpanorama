   int far vidmode(int newmode);

   Validate/set video mode.

 Parameters :

 newmode    -1 if you want to validate or new mode to set.

 Return :

 -1 if unsupported video mode was detected or set, or supported video
 mode (one of 0,1,2,3 or 7).

 Example :

 #include "window.h"

 void main(void)
 {
 int i;
     i=vidmode(-1);
     if (i==-1)
         printf("Unsupported video mode");
     else
         printf("Vidmode detected mode %i",i);
 }
