   void far _vputc(int x,int y,unsigned int char_attr);

   Write single character/attribute to screen.

 Parameters :

 x,y        Coordinates of screen location to write.
 char_attr  8 low bits are character, 8 hi bits are attribute.

 Return :

 None.
