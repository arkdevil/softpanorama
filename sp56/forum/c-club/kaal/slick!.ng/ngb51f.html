  As creating a menu isn't very easy, the number of parameters in function
 call would be enormous. Besides, most of these parameters never change
 in one program. So, why bother, let's put most of them into structure...
 The menu structure has following fields:

  o Flags indicating what to do with menu window and texts printed in it.
  o Pointer to 9 successive characters used to draw window frame.
  o Colours for window, normal items, selection bar, highlited characters.
  o Item number you want the menu bar initially to be on.

 Here is how it looks in SLICK!.H

 ───────────────────────────────────────────────────────────────────────────
 typedef struct {
     int makenew     :1;    /* 1 for open window, 0 if exist on screen */
     int center      :1;    /* 1 if you want items centered */
     int leavemenu   :1;    /* 1 if you don't want menu closed on exit */
     char far *frame;       /* frame for window */
     int wndcolor;          /* color for window (frame&fill) */
     char normcolor;        /* color for normal items */
     char invcolor;         /* color for selection bar */
     char hicolor;          /* color for hot keys */
     int defchoice;         /* item selected by default when entering */
     int reserved;          /* empty space on each side of item */
 } VMENU;
 ───────────────────────────────────────────────────────────────────────────
