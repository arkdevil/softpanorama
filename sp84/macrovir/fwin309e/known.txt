
The messages that F/WIN produces for the various KNOWN macro
viruses appear below.  If the infected file has a .DOC file
extension, an additional message will be produced that says:

- Contains macros but is named .DOC


          WordMacro.Boom

- Contains macros
- Contains macros but is named .DOC
- System macros: AUTOOPEN, AUTOEXEC, FILESAVEAS
- Contains execute-only macros (encrypted macros)
- Copy macros into the global template ('CopyMacro')
- Uses 'Organize .Copy' to copy macros
- Uses the 'FileSaveAs' macro command
- Disables NORMAL.DOT write access warnings
- Enables the quick save option
  Seems to be infected with a macro virus!
