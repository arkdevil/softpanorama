Windows Commander 3.01 16 bit fuer Windows 3.1 (laeuft auch unter Win95/NT)

Installation: 
Starten Sie einfach install.exe.

Sollte die Installation fehlschlagen, können Sie Windows Commander auch von
Hand installieren (siehe unten).

Was ist Windows Commander?

Windows Commander ist ein Dateimanager fuer Windows aehnlich wie der Windows-
eigene Dateimanager WINFILE.EXE bzw. Explorer. Windows Commander hat jedoch
2 Dateifenster nebeneinander, das das Kopieren von Dateien viel einfacher macht.
Wincmd gibt es in 2 Versionen: 16 und 32 bit. Waehrend die 16-bit Version
problemlos auch im 32-bit Windows laeuft, kann die 32-bit Version nicht unter
Windows 3.1 verwendet werden.

Windows Commander ist Shareware. Die Shareware-Version darf frei verbreitet
werden, sofern sie unveraendert bleibt!

Hier einige Eigenschaften (Features) von Windows Commander:

- Lange Dateinamen unter Windows 95 und Windows NT (16- und 32-bit Version).
- Unterstuetzt Drag & Drop (Ziehen und Fallenlassen) mit der Maus.
- 32-bit: Drag&Drop vom und zum Explorer und Desktop
- Verwendung Windows 95-spezifischer Merkmale wie Schnellansicht (quick view)
  und Papierkorb
- Erweitertes Kopieren, Verschieben, Umbenennen und Loeschen ganzer Verzeichnis-
  baeume auf einmal, mit Sicherheitsabfrage.
- Gepackte Dateien werden wie Unterverzeichnisse angesprochen. Dateien lassen 
  sich bequem in und aus gepackten Dateien kopieren, das entsprechende Pack-
  programm wird automatisch aufgerufen. Unterstuetzt werden pkzip, arj und lha,
  RAR und UC2.
- Starter-Menue (benutzerdefiniertes Menue).
- Konfigurierbarer Buttonbar.
- Kommandozeile zum einfachen und schnellen Starten von Programmen.
- Stark ausgebaute Suchfunktion mit Volltextsuche auch in gepackten Dateien.
- Eingebauter Dateibetrachter (F3), der Dateien beliebiger Groesse als Hex- oder
  Text darstellen kann, sowohl im ASCII- (DOS) als auch im ANSI- (Windows)
  Zeichensatz.
- Internes Unzip von Info-Zip
- Internes Un-ARJ und Un-LZH
- Interner ZIP-Packer

Die Share-Version von Windows Commander ist in keiner Weise eingeschraenkt,
bis auf einen Startbildschirm, der bei jedem Programmstart erscheint und auf
die Registrierung hinweist.

Installation VON HAND: Sollte die Installation aus irgendeinem Grund fehl-
schlagen, so benennen Sie die Dateien INSTALL.BIN und INSTALLD.BIN in 
INSTALL.ZIP und INSTALLE.ZIP um und entpacken Sie sie mit pkunzip -d in ein
Verzeichnis, z.b. c:\wincmd. Verschieben Sie dann alle Dateien mit der Endung
.pif in Ihr Windows-Verzeichnis! Dies bewirkt, dass die Packprogramme im
Fenster ablaufen. Die Datei ctl3dv2.dll gehört in Ihr Windows-SYSTEM-
Verzeichnis. Bitte löschen Sie alle älteren Versionen von ctl3dv2.dll.
CTL3DV2.DLL darf in keinem anderen Verzeichnis vorhanden sein!
