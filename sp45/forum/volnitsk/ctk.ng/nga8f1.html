 UnSetL ()
 Delete window displayed by Set Window()
 ─────────────────────────────────────────────────────────────────────────────────────

 Syntax

     UnSetWindow() --> NIL

 Description

     This function undo all actions made by last SetWindow(). That is
     restore screen area used by last window and delete Window Structure
     from Window List
See Also: Window System SetWindow()
