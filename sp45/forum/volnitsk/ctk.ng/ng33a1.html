 NotValidMsg()
 Display message at scroeboard
 ─────────────────────────────────────────────────────────────────────────────────────

 Syntax

     NotValidMsg ([cMsg])  --> .T.

 Arguments

     <cMsg>
          Charactor string storing message

 Returns

     NotValidMsg() always returns .T.

 Description

     Main goal of this function to be called from VALID expression of GET
     operator to display some massage at scoreboard area and beep error
     sound. This message remain at scoreboard until user take any action.


 Examples

     You must note that following example will work as expected only
     if compiled without /Z option.

     @ 10,10
          SAY      "Year of birth:";
          GET      nYear;
          PICTURE  "9999";
          VALID    (1900<nYear .AND. nYear<2000) .AND. NotValidMsg("Bad year")
