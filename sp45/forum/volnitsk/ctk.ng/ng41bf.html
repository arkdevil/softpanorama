 User interface functions

     CTK provide progammer with simple and at the same time flexible user
     interface functions. To make menu use Menu() or PromptBox().
     When using Menu() you can make menu with selectable items, variable
     length menu, and many other fancy menuing. PromptBox() best for
     bar menu. MsgBox() best for simple messages, messages with choices
     and simple hypertext. Any function can be called recursively.
