 Beep()

 ─────────────────────────────────────────────────────────────────────────────────────

 Syntax

     Beep(aSound, [lCondition]) --> NIL

 Arguments

     <aSound>
          List of fraquancy-duration pares defining some melody or complex
          sound flow.

     <lCondition>
          Optional ligical value. If missed then defaulted to .T.

 Returns

     Beep() always retuns NIL

 Description

     Used to play some complex sound defined in aSound. If current
     value of SET BELL is OFF then no sound are plaied. See also
     tone() function and SET BELL command.

 Examples

     We recomend to define all type of your sounds in some
     include file throw manifest constant. For example in
     include:

     #define   B_ERROR            {44, 3}
     #define   B_ALARM            {77,2,333,2}
     #define   B_PROCESS_END      {77,2,333,2,666,2}

     Later on in application you can just call:

     beep(B_ALARM)

     It is equivalent to:

     IF   set(_SET_BELL)
          tone(77,2)
          tone(333,2)
     ENDIF
