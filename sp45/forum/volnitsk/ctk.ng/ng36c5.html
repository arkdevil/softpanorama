 Color System

     This system is designed to make your projects attractive
     by using all available colors (color atributes) and simultaneously
     equaly well for common color and monohrome video modes
     and easily modified source code.

     All dialog objects of CTK (menu, messege box, input box and so on)
     use so named Color Stucture defining all there colors.
     One Color Structure element store color info about one dialog object
     element.

     Color Structure
     ────────────────────────────────────────────────────────────────
     CTK.CH         Value     Comment
     ────────────────────────────────────────────────────────────────
     xINTENS        1         Highlited color
     xPROMPT        xINTENS   Selectable item in MsgBox()/PromptBox
     xGET           xINTENS   Get area in InBox()
     xUNAVAILABLE   2         Used Menu()
     xHIGHLIGHTED   xINTENS   Used Menu()
     xHEAD          3         Window title
     xFRAME         4         Window frame color
     xTEXT          5         Common text, and available but not
                              selected items in Menu()
     ────────────────────────────────────────────────────────────────

     Color Structure can have NIL for not used elents (for example
     get color for MsgBox()).

     Color info include color definition for color and mono video
     modes in array with two corresponding charecter string contaning
     color attribute in SetColor() format.

     Color Structure element
     ────────────────────────────────────────────────────────────────
     CTK.CH         Value     Comment
     ────────────────────────────────────────────────────────────────
     CO80           1         color string for color video mode
     MONO           2         color sring for monohrome video mode
     ────────────────────────────────────────────────────────────────

     Current video mode stored in public xSet[xMODE] in form
     of numeric index to Color Structure element. Function InitCTK()
     initiate xSet[xMODE] with correct value. All CTK dialog objects
     use color string from Color Structure element according to video
     mode stored in xSet[xMODE].

     Color Structure can have NIL for not used elents (for example
     get color for MsgBox()).

     All Color Structures used in your application must be stored
     in public xSet [xCOLOR]. All CTK dialog objects functions have
     color parameter, which is subsequent index to xSet [xCOLOR]. So
     xSet [xCOLOR, nColorParm] is a Color Structure. Function InitCTK()
     when called create set of Color Structures which you can you use in
     your application.

     Windows (and therefor all dialog objects) in CTK have shadow.
     Shadow color (for two video mode) stored in public
     xSet[xCOLOR, xSHADOW, nVideoMode] which you can modify to your taste.

     To customise color of some dialog object you must create or modify
     some Color Structure by xSet() function. See also files CTK.CH and
     MSGBOX.PRG.
