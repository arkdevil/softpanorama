 SetList ()
 Set new or old values form Set List
 ─────────────────────────────────────────────────────────────────────────────────────

 Syntax

     SetList (aSetList, nType) --> NIL

 Arguments

     <aSetList>
          list (array) of new and old setting

     <nType>
          Set type stored in aSetList - new or old

          nType posible values
          ────────────────────────────────────────────────────────────────
          CTK.CH        Value      Comment
          ────────────────────────────────────────────────────────────────
          SL_NEW        2          set new values form Set List
          SL_OLD        3          set old values form Set List
          ────────────────────────────────────────────────────────────────

 Returns

     SetList() always retuns NIL

 Description

     Function SetList() used to handle Set List created by xSet().
     It's set new or old set values from Set List.

 Example

                                                       // BELL OFF
     local aList := {}                                 // SOFTSEEK OFF
     ...
     xSet ({_SET_BELL},     .T., aList)                // BELL ON
     xSet ({_SET_SOFTSEEK}, .T., aList)                // SOFTSEEK ON
     ...
     SetList (aList, SL_OLD)                           // BELL OFF
     ...                                               // SOFTSEEK OFF

     SetList (aList, SL_NEW)                           // BELL ON
                                                       // SOFTSEEK ON
See Also: xSet()
