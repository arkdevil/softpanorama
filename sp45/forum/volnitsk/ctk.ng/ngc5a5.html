 List2Array()
 Convert a delimited string to an array
 ─────────────────────────────────────────────────────────────────────────────────────

 Syntax

     List2Array (cList, [cDelimiter]) --> aList

 Arguments

     <cList>
          character string with field delinited by cDelimiter

     <cDelimiter>
          optional character repreasenting delimiter

 Returns

     ListAsArray() returns array each element is a field in cList
