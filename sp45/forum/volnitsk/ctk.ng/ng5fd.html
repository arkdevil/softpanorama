                  Clipper ToolsKit  1.0 beta

 What it is

     Clipper ToolsKit (or CTK) is library of functions which
     may simplify life of Clipper vertion not less then 5.01
     programmer. Its have:

     - set of functions for national language support, including: keyboard
       driver with changeable keyboard layout and any number of layout
       available at run time; changeable massages at scoreboard;
       changeble standart characters T/F at logical get fields to any other;

     - simple and flexible user interface functions;

     - Get System extentions;


 How to use

     Your source code must have #include "CTK.CH" directive and
     before you can use CTK, function InitCTK() must be called.
     To teminate application we recomend to use EOJ(). For using
     CTK you also must during linking to tell your linker search
     for functions in CTK.LIB file. In case RTLink, command line must
     include "LIB <PATH>\CTK.LIB" option. For example:

          rtlink  fi hello.prg  lib ctk.lib

     See also HELLO example and RUN.BAT
