 Translate()
 Translate string according to table
 ─────────────────────────────────────────────────────────────────────────────────────

 Syntax

     Translate      (cStr, cInTbl, cOutTbl)--> cTranslatedStr

 Arguments

     <cStr>
          Input chatacter string to be translated

     <cInTbl>, <cOutTbl>
          Parallel character strings which difines translation table.
          Must be of the same length.

 Returns

     Translate() return translated character string of the same length
     as cStr.

 Description

     All character in cStr that are found in cInTbl, are replaced with
     charecter from cOutTbl at the same position as found in cInTbl.

 Examples

     translate ('AaBbCc', 'abc', 'ABC')  -->  'AABBCC'
