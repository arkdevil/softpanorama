 EOJ()
 Terminate application and restore DOS screen
 ─────────────────────────────────────────────────────────────────────────────────────

 Syntax

     EOJ () --> nil

 Description

     Terminate application and exit to DOS. Used in pare with
     InitCTK(). Close all files, restore DOS screen and terminate
     application.
See Also: InitCTK()
