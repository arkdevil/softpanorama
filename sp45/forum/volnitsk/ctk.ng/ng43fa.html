 InBox()
 Make prompt box and input value
 ─────────────────────────────────────────────────────────────────────────────────────

 Syntax

     InBox (cTitle,  [nColor], [nGetVarMaxLen], [cInitVal]) --> cRecivedValue

 Arguments

     <cTitle>
          title of input box

     <nColor>
          optional Color Structure Index which specifies colors of all
          InBox() elements. See also Color System.


     <nGetVarMaxLen>
          optional maximam length recived value, if missed then maximum
          length equated to min (10, len(cInitVal))

     <cInitVal>
          optional inital value to be edited

 Returns

     InBox() return value recived from user

 Description

     InBox is simple function to enter character value. It desplay
     titled box in screen center and activate one field get. Then
     recived get value are retuned. If nGetVarMaxLen greater
     then screen dimetions you can scroll edited vlalue in get field.

 Examples

     Here InBox will prompt user to enter his name:

     cName := InBox ('Enter your full name', 50)
