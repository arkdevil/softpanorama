 GoToWindow  (nDestWindow)
 Move specified window to visible level
 ─────────────────────────────────────────────────────────────────────────────────────

 Syntax

     GoToWindow  ([nDestWindow]) --> NIL

 Arguments

     <nDestWindow>
          optional window number in Window List, first window
          has number 1.

 Description

     When you have to update not last window, this window can be
     overlaped by window displayed later. In such case you must
     first move needed window to visible level (ie top).
     If nDestWindow not specified then change will be made to
     window before last goto. If nDestwindow == 0 then all background
     will be visible and you can update it.

 Examples

     For example you need update some status window:

          ...
          GoToWindow (StatusWindow)     // goto status window

          // update status
          ...                           // update it

          GoToWindow ()                 // return to original window
