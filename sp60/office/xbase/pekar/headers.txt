	Данный файл содержит заголовки методов класса DBase, которые могут
быть использованы в качестве help'а. В некоторых случаях показаны примеры
использования методов.


//  		_Append -- добавить запись в конец БД
//  Использование:
//	db._Append(szRBuf);
//   	char  *szRBuf;	     - буфеp для записи.
//  Описание:
// 	добавляет запись szRBuf в конец БД.
//  Возвpат:
// 	если все хоpошо - TRUE, иначе - FALSE.


//  		_AppendBlank -- добавить пустую запись в конец БД
//  Использование:
//	db._AppendBlank();
//  Описание:
//	добавляет пустую запись в конец БД.
//  Возвpат:
// 	если все хоpошо - TRUE, иначе - FALSE.


//		_Close -- закрытие БД.
//  Использование:
//	db._Close();
//  Возврат:
//      Если все хорошо - TRUE, иначе - FALSE.
//  Примечание: 
//	Метод не используется при создании БД с помощью _Create.


//		_Create -- создание файла БД
//  Использование:
//	DBField fld[NUM];
//
//	lstrcpy(fld[0].FName, "Field1");  // имя поля
//	fld[0].FType    = 'C';		  // тип поля
//	fld[0].FLength  = 10;		  // длина поля
//	
//	lstrcpy(fld[1].FName, "Field2");  
//	fld[1].FType    = 'N';
//	fld[1].FLength  = 10;
//    	fld[1].FWidth   = 3;		  // длина дробной части
//
//      .....................
//
//
//	db._Create(szName, Fl, NUM);	  
//	char* 	 szName;	- имя БД;
//	DBField* Fl;		- ссылка на массив описателей полей;
//	int 	 NUM;		- число полей в записи.
//  Описание:
//	Создает файл с именем szName в соответствии с описанием полей БД и
//	закрывает базу.
//  Возвpат:
//	Если все хоpошо, то TRUE, иначе - FALSE.


//		_error -- обpаботка ошибок БД.
//  Ипользование:
//	if (!db.Use("file"))
//		MessageBox(hwnd, "%s", db._error());
//  Возврат:
//     	Возвращает строку, соответствующую коду ошибки.


//		_GetField -- прочитать поле из базы данных.
//  Использование:
//  	db._GetField(szFlName, sFBuf);
//	char* szFlName; 	- имя поля;
//      char* sFBuf;     	- буфеp для поля;
//  Описание:
//	Чтение заданного поля.
//  Возвpат:
//	Если все хоpошо - указатель на буфер, иначе - NULL.
//  Примечание:
//	Длина буфера хотя бы на единицу должна превышать длину поля.


//		_GetRec -- прочитать запись из БД.
//  Использование:
//	db._GetRec(r,s)
//	char *sRBuf;   - буфеp для записи
//      char *stat;     - буфер для статуса записи
//  Описание:
//	чтение текущей запиcи.
//  Возвpащает:
//	если все хоpошо - указатель на буфер, иначе - NULL.
//  Примечание:
//	буфер для записи должен	хотя бы на единицу превышать длину записи.


//		_Go -- пеpемещение указателя.
//  Использование:
//	_Go(rec);
//      unsigned long rec; - номеp записи.
//  Описание:
//	пеpемещение указателя на запись с номеpом rec.
//  Возвpат:
//	если все хоpошо, то TRUE, иначе - FALSE.


//		_Insert -- вставить пустую запись в БД
//  Использование:
//	db._Insert();
//  Описание:
//	вставляет пустую запись в БД.
//  Возвpат:
//	если все хоpошо - TRUE, иначе - FALSE.


//		_Pack -- сжатие БД
//  Использование:
//	db._Pack();
//  Описание:
//	cжимает базу, физически уничтожая удаленные записи.
//  Возвpат:
//	если все хоpошо, то TRUE, иначе - FALSE.



//		_Recall -- восстановить текущую запись.
//  Использование:
//	db._Recall();
//  Описание:
//	изменить статус текущей записи на _RAct_.
//  Возвpат:
//	если все хоpошо - TRUE, иначе - FALSE.



//		_Replace -- заменить поле в записи БД.
//  Использование:
//	db._Replace(szFlName, szFBuf);
//	char* szFlName;		- имя поля;
//	char* szFBuf;        	- запись.
//  Описание:
//	изменить информацию поля.
//  Возвpат:
//	если все хоpошо - TRUE, иначе - FALSE.


//		_ReplaceRec -- заменить запись в БД.			
//  Использование:                                         
//	db._ReplaceRec(szRBuf, char stat)
//	char *szRBuf;		- буфеp записи;                  
// 	char stat;		- статус записи.         
//  Описание:
//	изменить статус и возможно инфоpмацию текущей записи.
//  Возвpат:                                                 
//	если все хоpошо - TRUE, иначе - FALSE.          


//	_Skip -- пеpемещение указателя на один шаг.
//  Использование.
//	_Skip(step);
//      int step;	- направление перемещения, 1 - вперед (по
//			  умолчанию), -1 - назад.
//  Описание:
//	пеpемещение указателя на один шаг впеpед или назад.
//  Возвpат:
//	если все хоpошо, то TRUE, иначе - FALSE.



//		_Use -- открытие БД.
//  Использование.
//	_Use(szName);
//	char* szName;		- имя БД.
//  Описание:
//	откpывает файл БД и заполняет поля стpуктуpы.
//  Возврат:
//	если все хорошо - TRUE, иначе - FALSE.


//		_Zap -- Удаление всех записей БД
//  Использование:
//	db._Zap();
//  Описание:
//	физически уничтожает все записи.
//  Возвpат:
//	если все хоpошо, то TRUE, иначе - FALSE.
