  • Program Documentation
    • Commands
      • A - Add files to archive.
      • M - Move files to archive (erase files).
      • L - List files in archive.
      • X - Extract files from archive.
      • T - Test - Extract files to NUL.
      • D - Delete files from archive.
    • Options
      • /v - verbose
      • /a - maintain file attributes.
      • /s - collect/extract subdirs.
      • /o - no prompts.
      • /i - ignore EAs.
      • /e - save/restore empty directory & dir EAs.
      • /m - store modified files only.
      • /h - additional information.
