    Посмотреть  можно  как  с  использованием  накопителя  на
жестком диске, так и без него. В первом случае  надо  создать
на жестком  диске  каталог,  скопировать  туда  все  файлы  с
дискет, сделать этот каталог текущим. Во втором  случае  надо
вставить в карман дисковода  дискетку  и сделать этот  диско-
вод текущим. Дальнейшие действия одинаковы незасисимо от того
используется жесткий диск или нет:

    Для запуска рекламного ролика дать команду DEMO<Enter>
    Для чтения текста рок-оперы - JC<Enter>
    Для чтения художественных текстов - UV<Enter>

    Во время чтения можно использовать клавиши:

    Стрелки - Выбор
    Enter   - Переход к другому окну
    Home    - Переход к главному окну
    End     - Переход к предыдущему окну
