		Господа.
        Предлагаю Вашему вниманию переделку описания
PALа в гипертекст для Multi-Editа.
Для его инсталляции в Multi-Edit нужно:

1. Откомпилировать файл PARADOX.HLP командой

для Multi-Edit 5.00

HELPCMPX PARADOX
 полученный файл PARADOX.HLX записать в директорию
 Multi-Editа

для Multi-Edit 6.00 и 6.10
HELPCX PARADOX

 полученный файл PARADOX.HLZ записать в поддиректорию
 HELP директории Multi-Editа

2. для использования HELPа при работе со скриптами
3. войдите в меню INSTALL Multi-Editа в,
4. выберите раздел Falename extentions...
5. создайте раздел для расширения .sc
6. в графе Default Help File наберите PARADOX
7. запишите внесенные изменения.

        Теперь для вызова подсказки при редактировании
скрипта нужно набрать (F1 F9 - для Multi-Edit 5.00)
или (Ctrl-F1 - для Multi-Edit 6.00 и 6.10). Если курсор
при этом находился на ключевом слове PALа будет вызван
раздел в котором содержится его описание. Если данного
слова нет среди ключевых слов, будет вызвано общее меню).

				С уважением Поляк Л.Е. Москва 482-28-04
