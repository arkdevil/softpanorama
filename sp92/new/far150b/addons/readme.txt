"Archivers" folder contains command lines for some alternate to
used in archive support plugin archivers.

"Macros" folder contains useful keyboard macro commands.

"Tables" folder contains additional character tables, which can be
selected in FAR viewer and editor from menu, activated by Shift-F8.

To install any from the provided .reg files just press Shift-Enter
from FAR on its name and restart FAR.

All provided .reg files contain descriptions. Switch to "Long descriptions"
view mode to reead them.
