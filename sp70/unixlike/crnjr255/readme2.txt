
                             CronJr version 2.55

To do an upgrade from a previous version of CronJr do the following.

  1. Backup your CronJr.BAT, CronJr.SCH, Cron.EXE & EditCron.EXE files.

  2. Copy the Cron.EXE, TimeNow.EXE and EditCron.EXE files to their
     target directories.

  3. Create a new CronJr.BAT file by typing  CRON START  at the DOS prompt.

  4. To see the new features of CronJr, read the manual or just type
     cron  at the DOS prompt for a list of available parameters & features.


Please review the ERROR.LOG file for a listing of any known compatability
problems.


Look at the WINDOWS.DOC file for Windows 3.1 icon and pif support.
Look at the DESQVIEW.DOC file for DesqView support.
