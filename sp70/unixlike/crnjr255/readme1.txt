
                             CronJr version 2.55

New...
  New....

    1. Fixed 65 line entry limitation to CronJr.sch.
    2. Support for a network purge command to clean up temporary files.
    3. Fixed problem with F1 & Disable option causing immediate evaluation.
    4. Enhanced DesqView support via giving up unused time slices.

To print out the CronJr manual, type the following command at the DOS prompt.

C:\> copy manual.doc prn

To do a quick installation, type the following commands at the DOS prompt.

C:\> md cron
C:\> cd cron
C:\CRON> copy a:cron*.*
C:\CRON> cron start
C:\CRON> cronjr
