             Best Freeware Programs for DOS and Windows 3.1
             ==============================================

                                            Last updated 3 November 1996

This is a list of some freeware programs I like. As far as I know, all
are free for personal use. This includes public domain programs,
freeware, "postcard-ware", and shareware programs which are free for
personal or non-commercial use. It can also include demos, if they are
reasonably fully functional, can be freely used, and are not
time-limited. Some of these programs require payment if used for
commercial purposes.

Corrections and suggestions are welcome! Authors's suggestions are also
welcome, but please note that I only list free programs.

I update and post this list to Usenet groups alt.comp.freeware and
comp.archives.msdos.d about every two weeks. Programs new in this
version are marked "*NEW*".

I've marked a few programs I especially like with this sign: "*A TOP
PICK*".

The list is also available, with links to all the programs and to other
freeware resources, through the World Wide Web at:
     http://www.ncf.carleton.ca/~an448/

The latest version is usually available by FTP at:
     ftp://ftp.coast.net/Coast/msdos/info/free96??.zip
     ftp://ftp.winsite.com/pub/pc/win3/misc/free96??.zip
     ftp://ftp.elf.stuba.sk/pub/pc/text/free96??.zip

To get the list by E-mail, send a message to:
     infomart@acy.digex.net
with the subject:
     send freelist
Note that the subject must be exactly as shown, in lowercase.

Compiled by: Yves Bellefeuille <an448@freenet.carleton.ca>. PGP mail
welcome.

SECTION B:  COMPRESSION / DECOMPRESSION PROGRAMS AND UTILITIES

Quick guide:

      To decompress:             Use:
      ==============             ====

      *.arj                      ARJ
      *.com                      UNP
      *.exe                      UNP
      *.gz                       GZip
      *.lha                      LHA
      *.lzh                      LHA
      *.tar                      PCTar
      *.tar.Z                    First GZip, then PCTar
      *.tgz                      First GZip, then PCTar
      *.uue                      UU (see File and Text Utilities)
      *.z                        GZip
      *.zip                      Info-Zip
      *.zoo                      Zoo

ARJ - Supports multi-disk archives and self-extracting archives.
ftp://garbo.uwasa.fi/pc/arcers/arj250a.exe

*A TOP PICK* Diet - Compresses executable programs while leaving them
executable (similar to PKLite).
ftp://garbo.uwasa.fi/pc/execomp/diet145f.zip

GZip - To compress/decompress files using the Unix .Z format.
ftp://garbo.uwasa.fi/pc/unix/gzip124.zip

Info-Zip - PKZip compatible archiver, but doesn't support multi-disk
archives.
ftp://garbo.uwasa.fi/pc/arcers/unz520x.exe and zip21x.zip

LHA - Supports self-extracting archives.
ftp://garbo.uwasa.fi/pc/arcers/lha255e.exe

LZExe - Compresses executable programs while leaving them executable.
ftp://garbo.uwasa.fi/pc/execomp/lzexe91.zip

NCAV - Norton Commander Archive Viewer. Compression shell; lets you
view, edit, add and delete files, change compression method, etc.
Supports ARJ, LHA, Zip, Zoo, and other formats. Can be used as a
stand-alone utility, without Norton Commander.
ftp://ftp.coast.net/Coast/msdos/arcutil/ncav47.zip

PCTar - Unpacks files in the Unix "tar" format. Can convert Unix file
names to legal DOS names.
ftp://ftp.coast.net/Coast/msdos/archiver/pctar11.zip

UNP - Unpacks files compressed with PKLite, Diet, LZExe, EXECOMP, etc.
ftp://garbo.uwasa.fi/pc/execomp/unp411.zip

Zoo - An old archiver still used in the Unix world.
ftp://garbo.uwasa.fi/pc/arcers/zoo210.exe

How to get Garbo files by E-mail

For those without access to FTP or WWW, garbo.uwasa.fi runs an ftpmail
server which allows you to get Garbo files by E-mail. Please do not use
ftpmail if you can get the files by FTP or through the WWW.

The files will be sent to you in UUencode format. You'll need a UUdecode
program to decode them. If you don't already have one, you can get one
by E-mail as follows.

Send a message to:
     infomart@acy.digex.net
with the subject:
     send muud
Note that the subject must be exactly as shown, in lowercase.

Follow the instructions in the file you'll receive and you'll create a
basic UUdecode program called MUUD.COM. You may later want to get a more
complete UUdecode program, such as Udec, UU or UUdecode.

To get files from garbo.uwasa.fi, send a message to:
     ftpmail@garbo.uwasa.fi

The message's "Subject" can be anything at all. The ftpmail server will
simply ignore it.

The message should have the following contents:
     open garbo.uwasa.fi
     get [write here the full path and name of the Garbo file]
     quit

For example, to get the program Diet, send the following message:
     open garbo.uwasa.fi
     get /pc/execomp/diet145f.zip
     quit

Note that I'm giving the full path to the program, everything after
"ftp://garbo.uwasa.fi". Note also that I'm using slashes, "/", not
backslashes, "\".

You will get a confirmation message soon after your request. Depending
on the backlog, you may have to wait a few days before getting the file
itself. Please be patient, and don't keep requesting the same file!

To get more information on this service, send the following message to
ftpmail@garbo.uwasa.fi:
     help
     quit
