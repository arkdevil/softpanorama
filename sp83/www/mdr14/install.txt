Please be sure to read the README.TXT file which will be extracted after

============================
INSTALLING INTERNET MARAUDER
============================

Internet Marauder is distributed as either a self-extracting zip archive
called MDR14.EXE, or a regular zip archive called MDR14.ZIP.

The content of the two files is exactly the same.


To install from MDR14.EXE:

  1.  Create a temporary folder on your desktop.

  2.  Move the MDR14.EXE file into this temporary folder.

  3.  Double-click MDR14.EXE to run it.  It will extract the Internet Marauder
      setup program, SETUP.EXE, and related setup files.

  4.  Double-click the SETUP.EXE file in your temporary folder, and follow its 
      installation instructions.

  5.  Once setup is complete, you may delete the temporary folder and its 
      contents from your desktop.


To install from MDR14.ZIP:

  1.  Create a temporary folder on your desktop.

  2.  Move the MDR14.ZIP file into this temporary folder.

  3.  Use an unzip program such as WinZip or PKUnzip to extract MDR14.ZIP to this 
      temporary folder.  This procedure will extract the Internet Marauder
      setup program, SETUP.EXE, and related setup files.

  4.  Double-click the SETUP.EXE file in your temporary folder, and follow the 
      installation instructions.

  5.  Once setup is complete, you may delete the temporary folder and its
      contents from your desktop.
