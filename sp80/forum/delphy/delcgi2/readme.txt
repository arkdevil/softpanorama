TCGIEnvData (cgi.zip)   is Free.
TCGIEMail   (cgie.zip)  is $29.
TCGIDB      (cgidb.zip) is $39.

and a clear conscience.

(http://super.sonic.net/ann/delphi/cgicomp/install.html) and
and sample programs for each component available on-line.

The only official place to download these components is 
from http://super.sonic.net/ann/delphi/cgicomp/detail.html 
but you may set up mirror sites if you want.
