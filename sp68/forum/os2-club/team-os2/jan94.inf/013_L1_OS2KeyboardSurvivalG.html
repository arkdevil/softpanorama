Ever find that your mouse won't respond, but you need to do a proper shutdown before restarting your computer? Feel more comfortable with your hands on the keyboard rather than a small rodent? Here's a 4-part guide to help you get to know your keyboard better under OS/2. Let me know what you think of this. It looks OK on my monitor, but I'm curious if it works out for others. Enjoy!

Startup Tasks:

    Restore default desktop
    Inhibit auto-start features

OS/2 Command Line Tasks (with KEYS ON):

    Clear command line
    Move cursor to first character in command line
    Move cursor to last character in command line
    Turn on insert mode
    Delete character above cursor
    Move cursor one character to left
    Move cursor one character to right
    Display previous command in queue
    Display next command in queue
    Move cursor back to first character of a word
    Move cursor to first character of next word
    Deletes all characters to the left of cursor
    Deletes all characters from cursor to the end of the command line
    Executes the command on the command line

System Tasks:

    Switch to next window on desktop
    Switch to next window or full screen session
    Display the Window List
    Get help
    Switch to desktop
    Restart the system
    (twice) Dump system to diskette in A: drive (need diskettes created with CREATEDD command)
