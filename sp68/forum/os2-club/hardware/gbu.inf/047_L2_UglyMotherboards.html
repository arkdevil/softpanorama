    Comments
      Machine operates with AMD 486DX40 CPU run at 50MHZ, jumper selectable CPU speed, otherwise very stable
    Caveats
      System hangs and occasionally reboots when Mediavision PAS-16 uses 16-bit DMA channels, sometimes displays errors about not finding basic ROMS during 16-bit DMA induced reboot, PAS-16 sometimes works flawlessly if there is no hang on system boot-up. Not tested with more than 16MB of ram.
