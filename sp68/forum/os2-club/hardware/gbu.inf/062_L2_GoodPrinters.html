  • Type

      color ink jet
    Manufacturer
      Hewlett-Packard
    Model Number
      DeskJet 1200C
    Comments
      Latest HP drivers available from ftp-boi.external.hp.com; these drivers support color printing from OS/2 applications.
    Caveats
      Printer may run out of memory when printing large color images.
    Rated by
      RSMITH@PEARL.TUFTS.EDU - good

  • Type

      laser
    Manufacturer
      Hewlett-Packard
    Model Number
      LaserJet 4L
    Comments
      Latest HP drivers available from ftp-boi.external.hp.com. Printer is supported directly by OS/2 out of the box.
    Caveats
      Printer jams more frequently than it should. Jams intensify if double-sided (two-pass) printing is attempted from a program which supports this (such as DeScribe).
    Rated by
      RSMITH@PEARL.TUFTS.EDU - good

  • Type
