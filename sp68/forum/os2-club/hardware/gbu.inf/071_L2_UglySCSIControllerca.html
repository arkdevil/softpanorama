Ugly SCSI Controller cards

----

Type

    Fast SCSI-II, Floppy controller, ISA Bus
Manufacturer
    Future Domain
Model Number
    TMC-1680 (SVP Package)
Chip Set
    Bios v3.2
Comments
    Driver included with OS/2. Updated drivers available from Hobbes FTP or Future Domain. Jumpered settings for:

    o

      IRQ's 3,5(d),10,11,12,14,15
    o
      Memory Address' C800,CA00(d),CE00,DE00
    o
      I/O Address' 0140(d),0150, 0160,0170
    o
      Floppy On(d)/Off Has Micro DB50 connector (SCSI-II connector)
    Caveats
      Will not boot OS/2 v2.1, or Warp Beta I/II, off SCSI drive if set to IRQ 15, yet will happily run and access these drives if booted off of IDE drive. (Will boot DOS/Windows, Win NT, Win 95 Beta at IRQ 15.) IRQ14 and IRQ5 both allow OS/2 to boot off SCSI drive. Others not tested.
