42h - Set Line Characteristics

Parameter Packet 

     ┌──────────────────────────────────────┐
     │         Item                Type     │
     ├───────────────────────────┬──────────┤
     │ Data Bits Per Character   │ BYTE     │
     ├───────────────────────────┼──────────┤
     │ Parity Setting            │ BYTE     │
     ├───────────────────────────┼──────────┤
     │ Stop Bit Setting          │ BYTE     │
     └───────────────────────────┴──────────┘

Data Bits Per Character may be 5, 7, or 8, all other values are invalid.

Parity Setting may have the following values, all other values are invalid: 0 - No parity bit.
1 - Odd parity.
2 - Even parity.
3 - Mark parity (1).
4 - Space parity (0).

Stop Bit Setting may have the following values, all other values are invalid: 0 - 1 stop bit.
1 - 1.5 stop bit (valid with 5 data bit characters).
2 - 2 stop bits.

Data Packet - None
