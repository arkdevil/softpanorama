Installing ZipStream is easy. You just use the 'Device Driver Install' feature that comes with OS/2.

To get started, proceed as follows:

  • Insert the ZipStream diskette in your A: drive.

  • Now open your 'OS/2 System' folder

  • Then open the 'System Setup' folder

  • Start the 'Device Driver Install' utility

  • Hit the 'Install' button, and wait for the ZipStream installation to complete.

  • After the installation is complete, hit the 'Exit' button.

  • The 'Device Driver Install' utility will inform you that you must now reboot for the changes to take effect.

  • Shutdown your system in the standard way (i.e. from the Desktop, DO NOT Ctrl-Alt-Del).

    When your system restarts, the ZipStream File System will be loaded. You will then be able to compress files and access them again transparently.
