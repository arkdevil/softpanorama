Introduction

The ZSAttach utility is used to:

    o

      Make the ZipStream connection with other file systems.
    o
      Display current ZipStream connections.
    o
      Cancel current ZipStream connections.

    Usage

      ZSAttach ZipStream_Drive OS/2_Path

        ZipStream_Drive may be any free or unassigned OS/2 drive letter.

        OS/2_Path may be any valid OS/2 directory.

        This operation establishes the ZipStream connection with an OS/2 directory.

        Example

        You wish to attach the root directory of the C: drive as the ZipStream H: drive. You enter at the command prompt:

        zsattach h: c:\

      ZSAttach ZipStream_Drive

        ZipStream_Drive must be a ZipStream drive letter assigned to an OS/2 Directory.

        This operation reports the current OS/2 Directory path assigned to the ZipStream drive letter.

        Example

        You have a ZipStream drive H: and you wish to determine the directory to which it is attached. You enter at the command prompt:

        zsattach h:

        ZSAttach reports:

        h: ZipStream drive connected to c:\

      ZSAttach ZipStream_Drive /D

        ZipStream_Drive must be a ZipStream drive letter assigned to an OS/2 Directory.

        This disconnects the ZipStream connection with an OS/2 directory.

        Example

        You have a ZipStream drive H: connected to the root directory of drive C: and you wish cancel the connection. You enter at the command prompt:

        zsattach h: /d

      ZSAttach

        This operation reports all current ZipStream connections with OS/2 Directory paths.

        Example

        You have a ZipStream drive H: connected to the root directory of drive C: and ZipStream drive G: connected to D:\. You enter at the command prompt:

        zsattach

        ZSAttach reports:

        h: ZipStream drive connected to c:\

        g: ZipStream drive connected to d:\
