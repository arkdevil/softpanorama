NETSCAPE AUTOADMIN

CENTRALIZED AND AUTOMATED ADMINISTRATION AND MANAGEMENT OF NETSCAPE COMMUNICATOR CLIENTS

Managing the installation and configuration of client software is expensive and time-consuming for IS departments. Netscape AutoAdmin (available in Netscape Communicator Professional Edition) allows IS managers to centrally manage and update Communicator. To set up and adminster AutoAdmin, an IS manager uses the tools provided in Netscape Mission Control.

Set and change client settings from a centralized server.
  • AutoConfig provides centralized management of more than 200 client settings.
  • Settings can be managed from a server-side file or through any LDAP-compliant directory server.
  • Configurations can be easily extended using JavaScript.

Update Communicator clients automatically with new settings or buttons.
  • Communicator supports automated polling and download of new client settings, buttons, and menus.

Automatically download, or "push," new versions of Communicator components and third-party add-ons.
  • AutoUpdate enables Communicator to automatically download and install new components or third-party plug-ins.
  • Using Mission Control, components can be digitally signed with a corporate or organizational digital certificate.
