NETSCAPE MESSENGER

FULL-FEATURED, OPEN STANDARDS-BASED EMAIL

Netscape Messenger is a full-featured application for sending and receiving email over intranets or the Internet. Messenger includes new message-management features that are powerful enough for corporate use but easy enough for individual use. Open standards-based features, such as mobile user support and online address searching, allow users to access their email from any location and find people anywhere on the Internet.

Messenger is fully integrated with Netscape Composer, allowing users to create email messages that look like web pages, complete with graphics, images - even Java applets. As with all other Communicator components, Messenger includes encryption to protect private communications.

Send images, graphics, and Java applets in an email message.
  • Support for HTML mail allows Messenger to send email messages that look and behave like web pages.
  • Full integration with Composer's HTML editing allows users to mail Composer documents.

Encrypt personal and confidential email.
  • Support for S/MIME provides data encryption, which protects messages as they are transmitted over the Internet, and digital signatures, which positively identify message authors to recipients.

Send email to and receive mail from other Internet mail applications.
  • Support for standards, including IMAP4, SMTP, POP3, MIME, UUENCODE, and BINHEX, means that Messenger users can read virtually any email message and send messages to anyone on the Net.

FIND EMAIL ADDRESSES EASILY AND QUICKLY, AND SAVE THEM IN A PERSONALIZED ADDRESS BOOK
Messenger's Personal Address Book is a full-featured tool for organizing electronic addresses of all kinds.
Support for the LDAP directory standard allows Messenger to search any open standards-based online directory.
Users can personalize messages with vCard digital business cards.

CHECK SPELLING IN DRAFT EMAILS BEFORE SENDING
Fully integrated spelling checker helps eliminate spelling errors.

ORGANIZE AND PRIORITIZE INCOMING MESSAGES
Define priorities and sort messages with filters.
Access all message-based communication from the Communicator Message Center.

ACCESS AND MANAGE MESSAGES FROM MULTIPLE LOCATIONS
IMAP4-based message management allows all incoming messages to be read anytime, anywhere from IMAP4-compliant mail servers.
