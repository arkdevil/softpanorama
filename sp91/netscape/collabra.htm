NETSCAPE COLLABRA

OPEN STANDARDS-BASED COLLABORATIVE GROUP DISCUSSION FOR INTRANETS AND THE INTERNET

Netscape Collabra brings the benefits of newsgroups and online information exchange to workgroups, businesses, and corporations. As easy to use as email, Collabra creates online discussions for collaborating on key issues or soliciting input from colleagues, reducing - or even eliminating - the need for face-to-face meetings.

Collabra's open standards-based discussions can also be integrated with Internet newsgroups (online discussions between Net users on specific topics). Powerful information-management tools allow users to prioritize postings, or messages, and eliminate irrelevant information. Like all other Netscape Communicator components, Collabra includes open standards-based, high-level security features.

Start online discussions with colleagues or join Internet newsgroups.
  • Support for NNTP newsgroup communications standards allows Collabra discussions to link to any open standards-based newsgroup.

Find relevant internal discussions quickly and easily.
  • Collabra allows user-friendly names for newsgroups.
  • Collabra provides categorization for discussions.
  • Users can perform cross-discussion searches for relevant information.

Prioritize and manage discussion postings.
  • Collabra's filters allow users to highlight certain postings and ignore others.
  • Manage postings from Communicator's Message Center.

Post forum messages with graphics, images - even Java Applets.
  • Documents created in Composer's HTML editing environment can be displayed and viewed in a Collabra forum.

Create private internal discussions.
  • Collabra supports Netscape's SSL 3.0 security standard.
  • Specific groups of users can be defined with access control.
