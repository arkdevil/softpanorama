NETSCAPE NAVIGATOR 4.0

THE PREMIER INTRANET AND INTERNET BROWSING TOOL

Netscape Navigator 4.0 is the newest version of Netscape's world-leading software for browsing information on intranets or the Internet. Navigator has been updated and improved for greater ease of use. Plug-ins (small programs that add functionality to Navigator) are now downloaded and installed automatically so they work right away without any user configuration. New information-management tools simplify access to favorite sites and provide one-click access to Internet searches.

Powerful new features and technology allow developers to create visually exciting, fully interactive pages. Users can directly manipulate and move images and Java applets.

Like all Netscape Communicator components, Navigator includes open standards-based security features to protect information such as credit card numbers and personal financial data.

Customize Navigator 4.0 to access intranet and Internet information the way you want.
  • Navigator 4.0's toolbar can be customized to individual preferences.
  • Bookmarks for marking favorite sites are more powerful and easier to use.

Get simple and quick access to the Internet's most powerful search engines.
  • The new Search button links directly to a collection of the Internet's leading search engines.

Access Netscape Guide By Yahoo with a single click.
  • The Places button takes users directly to this Internet site directory.

Encrypt sensitive data.
  • Netscape's SSL 3.0-based security is integrated throughout Navigator 4.0.
  • Data encryption in Navigator 4.0 protects information as it is transmitted over the Internet.

Access dynamic, interactive, and visually rich sites.
  • Dynamic HTML creates visually rich pages that are faster and easier to access.

View and listen to multimedia information without becoming a multimedia expert.
  • Navigator 4.0 comes pre-installed with the most popular multimedia plug-ins.
  • SmartUpdate automatically configures and installs new multimedia plug-ins.
  • Supports interactive 3D with the Cosmo Player VRML 2.0 rendering engine.

Access other documents, such as Microsoft Excel spreadsheets, from Navigator 4.0.
  • Support for ActiveX documents (OLE) allows Navigator 4.0 users to link directly to other ActiveX applications, such as Microsoft Office, from within a Navigator window.
