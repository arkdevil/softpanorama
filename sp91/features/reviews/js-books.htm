NNB Recommended Javascript books

  • How to Program HTML Frames: Interface Design with JavaScript; Teague, Jason

    • Ziff - Davis; 05/97; $35.99 (10% off list);

    This is one of the few non-beginners books. It presuppose that you already know how to use Javascript.

  • Java 1.2 And Javascript For C And C++ Programmers; Daconta, Mike

    • John Wiley & Sons; 05/97; $44.99;
  • JavaScript Bible, Second Edition; Goodman, Danny

    • IDG Books Worldwide; 11/96; $35.99 (10% off list).

    Author have WWW page for the book with some examples available online. Such a book is always more useful than book without supporting WEB site. Book is probably one of the most useful beginner books on the subject.

  • Javascript Manual of Style; Johnson, Marc

    • Ziff Davis Press; 06/96; $22.49 (10% off list); Ships same day

    Author has very intresting home page. He also author Assembler programming book.

  • Javascript Quick Reference : Forms, User Events, Browser Controls; Darnell, Rick

    • QUE; 08/96; $17.99 (10% off list);
  • JavaScript: The Definitive Guide, Second Edition; Flanagan, David

    • O'Reilly; 02/97; $29.65 (10% off list);

    Highly recommended on the WEB book. Considered as the best by developer of Javascript and Jscript.
