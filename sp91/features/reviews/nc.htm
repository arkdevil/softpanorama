The First Impressions on Symantec Norton Commander for Windows95/NT

by Nikolai Bezroukov

Introduction

The first impressions are positive. Generally is can be used as a Window clone of NC. Pretty slim if you compare it to this fatty Navigator 2 for Win95 (distribution is on 2 floppies, see below). Scheduler proved to be a pretty useful addition. But file manipulation capabilities and support of command line are still underpowered in comparison with Volkov Commander.

Achievements

  • Level of compatibility with the DOS NC v.5 is decent. So users of old good NC will feel themself more or less at home
  • Select file filter at last has history. In addition to file filters via "Grey +,-,*" it has operation filters that permit more flexible copying/moving of files;
  • NCW can mount/unmount network drive;
  • Find option is greatly improved and contain possibility of using real regular expressions and finding duplicate files;
  • Have pretty useful scheduler. I like very much that it plays chime every hour.
  • Keyboard compatibility is very good. Some additional capabilities and problems are present due to Windows env.:
  • Shift-F2 provide NC Network neighborhood panel. Pretty useful !
  • You should use Ctrl-Alt instead of Alt for quick search in panel.
  • Right click on the mouse select the file, which is not so useful in NT env. (IMHO menu on right click is a better way). You can change it via Commands/Configuration, but is will highlight selected file anyway;
  • Shift-F2 launch Norton Utilities if they are installed.
  • Ctrl-F10 split of merge selected files.
  • Understand long file names and is NTFS compatible

Shortcomings

  • Very expensive ($120). So it seems to be overpriced 5 times or so :-).
  • It does not supply its own viewers and uses basic Quick View viewers. So to view HTML one need to get full version of Quick View. If you try to use Navigator as a HTML viewer NCW try to display additional non existent empty file after loading the original one (may be extra keystroke is transmitted to Navigator, or something like that);
  • No history exists for directories displayed in left-right panels;
  • Ctrl-E, Ctrl-R and Crtl-O do not work at all. Generally command line support leave much to be desired (no auto switch to DOS screen after execution of DOS command, etc. No quick entering of left and right panel path into command like in Volkov Commander.
  • Macro support via F2 and file extension is stone age like. Volkov Commander do much better job than $120 commercial product
  • Renaming of directories is clumsy.

Recommended changes in the Configuration:

  • Suppress confirmation on copy operation in Command/Configuration/General;
  • Enable the following items in Command/Configuration/Screen:
    • Tab bar;
    • Tools bar.
  • Disable Key Bar. IMHO it does not provide any useful functionality for advanced users
  • Specify in Command/Configuration/Compatibility "Right Click Pop Up Menu"
  • Specify different external editor (Qedit, Kedit or whatever you like best).

Installation

Distribution consists of two floppies. It asks several things like directory and colors: one can choose classical blue NC colors or Windows like colors.
