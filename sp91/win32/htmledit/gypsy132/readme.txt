Installation notes:

gypsy is self-contained within its own folder.

To set the default screen font and default test browser:
1. Open gypsy, but do not open any documents.
2. On the top menu select File | Browser and locate your browser 
(for example: netscape.exe)
3. On the top menu select File | Setup | Screen Font
4. Select the font you would like to use in the gypsy editor.

Note that custom tags are also added in this manner by selecting 
File | Setup | Custom tags when no documents are open.

---
Creating a gypsy icon for the desktop or start menu:

1. Using Explorer, locate gypsy01.exe in the gypsy folder.
2. Right click on gypsy01.exe, select create shortcut.
3. Right click on the shortcut, select cut or copy.
4. Paste the shortcut on your desktop or in the appropriate start 
menu folder.

Note that by right clicking the shortcut, you can select Properties 
and change the default directory for gypsy to use on startup.

---
Removing gypsy:

Simply delete the gypsy folder from your hard drive. gypsy does 
not place any files anywhere else on your hard drive.

---
Differences between free gypsy and registered gypsy versions:

Free gypsy is fully functional software.

The registered version of gypsy includes three additional features
which are not available in free gypsy: a spell checker, import
of tab delimited and rich text files and the ability to view gif
format graphics. For information on the registered version of
gypsy see the gorder.txt file.
