Here you can download the whole Introduction to JavaScript tutorial. I have prepared several versions of my tutorial. First there is the HTML-version which is available as a zip-archive and a tar-archive. Then there are several printable versions - a ps-version (postscript), a pdf-version (Adobe Acrobat) and a MS Word 7.0 version.

The HTML-version

  • jsintro.zip (ca. x kB - last updated: May 17th, 1997) - you'll have to unpack this file with a program like PKUnzip. The zip-archive contains several subdirectories.

  • jsintro.tar.gz (ca. x kB - last updated: May 17th, 1997)

The printable version

  • jsdoc.zip (ca. x kB - last updated May 17th, 1997) - this is a zip-archive. You'll need a program like PKUnzip to unzip it. The archive contains several MS Word 7.0 files.

  • jsps.zip (ca. x kB - last updated May 17th, 1997) - This is the postscript-version of the tutorial (zip-archive).

  • jsps.tar.gz (ca. x kB - last updated May 17th, 1997) - This is the postscript-version of the tutorial (tar-archive).

  • Here is the pdf-version of my tutorial (last updated May 17th, 1997). You can view these files for example with the Adobe Acrobat Viewer which is available for free.
