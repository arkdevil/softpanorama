        Программисты, работающие на FoxPro !  Вашему вниманию предла-
гается набор инструментальных средств R_FOX KIT, предназначенный для
функциональной руссификации FoxPro 2.0.
        R_FOX KIT поможет вам решить  проблемы  ввода  с  клавиатуры
русской буквы 'Н' и корректной индексации/сортировки  по  алфавитным
русскоязычным полям.
        В комплект поставки R_FOX KIT входят следующие файлы:
     README.TXT   - краткое описание и прайс-лист ( этот файл вы чи-
                    таете в данный момент).
     RFOX_DOC.TXT - техническое описание и инструкция по использова-
                    нию R_FOX.KIT
     R_FOX.COM    - резидентный  драйвер,  решающий  проблему  ввода
                    русской 'H'( совмещенные Демо и Рабочая версии )
     R_FOX.MEM    - кодировочная таблица,используемая для индексации
                    по русскоязычным полям с чувствительностью к ре-
                    гистру
     R_FOX_IC.MEM - кодировочная таблица,используемая для индексации
                    по  русскоязычным  полям без  чувствительности к
                    регистру
     RFGUID01.PRG - демонстрационный пример, показывающий различия в
                    в способах индексации с  использованием  кодиро-
                    вочных таблиц R_FOX и R_FOX_IC
     RFGUID02.PRG - пример программы, использующей кодировочные таб-
                    лицы R_FOX и R_FOX_IC для создания индексов.
     DICT.DBF     - файл данных, используемый демонстрационными при-
                    мерами
     DICT.CDX     - индексный  файл, используемый  демонстрационными
                    примерами


                КРАТКОЕ ОПИСАНИЕ R_FOX KIT.

       Функциональная руссификация FoxPro 2.0 обеспечивается за счет
резидентного драйвера R_FOX.COM, загружаемого перед запуском FoxPro,
и двух кодировочных таблиц R_FOX и R_FOX_IC, загружаемых  в  динами-
ческую память FoxPro командой RESTORE.
       Дополнительный расход памяти составляет:
        - 1 Кбайт на резидентный драйвер
        - 290 байт на каждую из кодировочных таблиц.
       Использование R_FOX KIT не увеличивает  размер индексных фай-
лов; при тестировании не было замечено замедление  выполнения команд
индексирования, сортировки и индексного поиска.
       R_FOX KIT не меняет  кода задач, входящий в FoxPro, что обес-
печивает соблюдение авторских прав фирмы  Microsoft  и совместимость
с последующими версиями FoxPro.
       Благодаря оригинальной технологии отслеживания  запуска задач
(см. Техническое описание и инструкцию), драйвер R_FOX.COM активизи-
руется только на период выполнения приложений, созданных  с  помощью
FoxPro ( файлы *.PRG, *.APP, *.EXE-compact и *.EXE-standalone ). Для
всех остальных задач драйвер прозрачен и не требует  выгрузки из па-
мяти.
