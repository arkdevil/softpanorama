Client/Server Pack

Технология клиент-сервер теперь доступна программистам и
пользователям IBM PC! 

Впервые программисты и пользователи IBM PC получили в составе
одного продукта все необходимое для построения приложений с
архитектурой клиент-сервер. Client/Server Pack включает в себя
как  мощные и гибкие средства Paradox 4.5 Development Edition
для Windows, предназначенного для использования в качестве
интерфейсной части приложений, так и сервер баз данных InterBase
спроектированный по наиболее передовой технологии построеня
реляционных баз данных.

Со времени широкого внедрения технологии локальных
вычислительных сетей переход к архитектуре клиент-сервер
является наиболее заметным шагом вперед в индустрии обработки
данных. С появлением Сlient/Server Pack переход на новую
технологию стал не сложнее простого копирования файлов данных с
файл-сервера на сервер баз данных. Существенно, что теперь,
также впервые, весь набор средств для построения систем с 
архитектурой клиент-сервер доступен через стандартные каналы
распространения программного обеспечения.

Paradox 4.5 Development Edition для Windows включает в себя
целый ряд дополнительных средств, существенно упрощающих процесс
создания клиент-сервер приложений, в первую очередь Borland SQL
Link для доступа к SQL-данным.

InterBase - технологически наиболее современное ядро для
построения приложений с архитектурой клиент-сервер и
распределенных баз данных. Традиционно функционируя на всех
основных UNIX-платформах, InterBase в ближайшее время будет
доступен в средах Windows NT, Novell NetWare и IBM OS/2

Client/Server Pack 

Идеальный и экономически наиболее оправданный выбор для
разработки приложения с архитектурой клиент-сервер. В состав
пакета включена версия InterBase для SCO UNIX с лицензией на 2
пользователя, а так же Paradox 4.5 Development Edition для
Windows.

Paradox 4.5 Development Edition для  Windows 

 Наиболее мощный пакет семейства Paradox 4.5  для Windows. Кроме
полного пакета Paradox 4.5 Workgroup Edition  для Windows этот
вариант содержит дополнительные средства для плавного перехода к
архитектуре клиент-сервер.

Paradox 4.5 Workgroup Edition для Windows



Содержит полную версию Paradox 4.5 для Windows и ObEx -
программное обеспечение для коллективной работы. Кроме того, в
состав пакета включены драйверы для доступа к трем наиболее
популярным SQL-серверам баз данных InterBase, Oracle, Sybase и
средства для работы с удаленными SQL-таблицами.

Paradox 4.5 Development Tools для Windows

Дополнительные средства для разработчика, создающего приложения
в масштабах крупного предприятия с возможностью доступа к
SQL-серверам.

Resource Workshop - средство создания ресурсов приложений для
Windows, позволяющее придать профессиональный вид вашим
приложениям.

Help Compiler - компилятор справочных файлов для создания
дружественных приложений.

Исходные тексты SQL Tools - средств интерактивного
взаимодействия с SQL-сервером являются хорошим примером
реализации такого взаимодействия и служат средством освоения
новой технологии создания приложений.

Developer's Toolkit - набор готовых  объектов для использования
в формах и окнах диалога.



Сервер Баз Данных InterBase

InterBase - это система управления реляционными базами данных
для построения приложений с архитектурой клиент-сервер,
поставляемая корпорацией Borland для компьютеров различных
классов: IBM PC под управлением SCO UNIX, рабочих станций и
серверов Sun, Hewlett-Packard, IBM, мини-компьютеров VAX и др. В
качестве интерфейсных среств InterBase могут быть использованы
хорошо знакомые пользователю продукты Borland: Paradox, Quattro
Pro, Borland C++ и др. InterBase обеспечивает: хранение внутри
базы данных неструктурированной информации (изображение, звук и
пр.), наличие активного ядра, устраняющего необходимость
непрерывного опроса базы данных, ведение нескольких поколений
записей, уменьшающее конфликты чтения и модификации одной и той
же записи и другие возможности, уникальные среди современными
СУБД.

Среди пользователей InterBase крупнейшие корпорации и мощные
торгово-финансовые объединения разных стран такие, как Boeing,
McDonnel Douglas Astronautics, Boston Stock Exchange,
Philadelfia Stock Exchange, а также правительственные органы США
и др.



Paradox 4.5 для Windows - новая версия ведущей реляционной СУБД
для Windows. Paradox поддерживает форматы файлов, индексы и
MEMO-поля dBASE и FoxPro и может работать с базами данных на
SQL-серверах InterBase, Sybase, Microsoft и Oracle. Режим
запроса по образцу используется для формирования запросов не
только по локальным, но и по удаленным данным на SQL-серверах
БД. В дополнение к стандартным типам данных в Paradox для
Windows можно использовать поля комментариев произвольной длины
(MEMO-поля), графические поля, а также произвольную информацию
типа оцифрованного звука в полях типа BLOB (большие двоичные
объекты). Панель управления с контекстно-чувствительными
кнопками и меню Инспектора Объектов позволяют за несколько
щелчков мышью изменить любую характеристику текущего объекта.

Quattro Pro 5.0 для Windows - Сочетание легкости использования и
мощных возможностей выводит новую версию Quattro Pro в лидеры
рынка электронных таблиц для среды Windows. Подсистемы обучения
и контекстно-зависимой помощи дополнены функционально-полным
набором средств анализа и моделирования данных. Сочетание
средств доступа к SQL-серверам баз данных со встроенной
технологией коллективной обработки данных обеспечивает
качественно новый уровень Quattro Pro for Windows. Более 20
программ-Экспертов автоматически выполняют за пользователя
наиболее сложные операции над обрабатываемыми данными, а
Менеджер сценариев позволяет строить множественные сценарии по
принципу "Что, если ?".
