 Power made easy   

	

компания Borland объявляет                                      
      четыре новых версии Paradox

Объявлены новые версии ведущей реляционной системы управления
базами данных Paradox для операционных систем DOS и Windows.
Новые продукты Paradox 4.5 для Windows, Paradox 4.5 для Windows
Workgroup Edition, Paradox 4.5 для Windows Development Edition и
Paradox 4.5 для DOS включают такие усовершенствования, как
сопряжение с SQL-серверами баз данных, технологию коллективной
обработки данных и новые расширения возможностей
программирования.

Новые версии соответствуют стратегии компании Borland - поставка
продуктов трем основным категориям пользователей: работающим
дома и в небольших офисах, корпорациям и разработчикам
программного обеспечения. Paradox 4.5 для Windows - простая в
эксплуатации, полная по составу характеристик СУБД, цена которой
соответствует возможностям индивидуального, малого и среднего
бизнеса. Для корпоративных пользователей, имеющих прикладные
задачи, решаемые на основе архитектуры клиент/сервер и
нуждающихся в возможностях групповой работы, предложена версия
Paradox для Windows Workgroup Edition, построенная на базе
пакета Paradox и предлагающая специальные решения для
корпоративных приложений. Версия Paradox Development Edititon
предоставляет инструменты, предназначенные специально для
построения и развертывания  прикладных программ в системе
клиент/сервер в рамках предприятия в целом.

"Со времени представления первой версии пакета Paradox для
Windows, на международном рынке баз данных для среды Windows
сложился образ компании Borland как лидера в этой области," -
сказал Дэвид Уоткинс, вице-президент и руководитель отдела
управления и маркетинга продукта (Applications Business Unit)
компании Borland. "Новейшие версии семейства Paradox
обеспечивают пользователей баз данных лучшими решениями
широчайшего спектра приложений. Предлагая Paradox, компания
Borland дает пользователям прозрачную совместимость версий для
сред DOS и Windows. Зная, что Paradox - единственная СУБД для
ПК, предлагающая пользователю стопроцентную совместимость
данных, корпорации уверены в сохранности своих инвестиций.
Семейство продуктов Paradox предлагает полные, расширяемые
возможности базы данных, начиная с отдельных продуктов и вплоть
до сопряжения с SQL-сервером баз данных и групповой работы, чего
не может обеспечить ни одна СУБД для среды Windows,
представленная сегодня на рынке."

Paradox 4.5 для Windows Paradox для Windows - идеальный пакет
для малого и среднего бизнеса. Он предлагает простые в
эксплуатации возможности для новых пользователей, гибкость,
необходимую опытным пользователям и инструментальную среду
разработки, необходимую программистам для построения изысканных
прикладных программ. Для обеспечения более широкого контроля со
стороны пользователя за внешним видом форм и отчетов добавлены
настраиваемые таблицы стилей. Теперь Paradox для Windows
поддерживает форматы файлов, индексы и MEMO-поля пакета FoxPro,
так же как файловые форматы dBASE. Улучшение производительности
и новые команды ObjectPAL дают пользователю еще большие
возможности по созданию качественных приложений.

Paradox 4.5 для Windows Workgroup Edition Версия Paradox
Workgroup Edition - первая система управления базами данных для
среды Windows, предназначенная специально для групповой работы,
с помощью технологии Обмена Объектами, предлагаемой компанией
Borland. Обмен Объектами позволяет пользователям одновременно
работать с информацией из разных приложений или с одним и тем же
приложением.

Workgroup Edition обеспечивает доступ к данным, расположенным на
различных удаленных серверах баз данных, включая InterBase 3.3,
ORACLE SQL Server, Microsoft и SYBASE SQL серверы. С помощью
Borland SQL Link, пользователи могут работать с данными
SQL-сервера так, как если бы они были локальными, и в диалоговом
режиме, и из программ на ObjectPAL.

Утилита Workstation Configuration Utility позволяет
администратору сети устанавливать в сеть Paradox Workgroup
Edition и выбирать опции, задаваемые по умолчанию для каталогов
и значков, которые затем будут установлены на рабочей станции
пользователя сети. Пользователю достаточно запустить утилиту
конфигурации, ввести свое имя и название компании, чтобы начать
работать с Paradox. Кроме того, поставляется утилита,
позволяющая пользователю при желании изменять значения,
назначенные по умолчанию.

Таблицы стилей Paradox значительно расширяют возможности
контроля со стороны пользователя за стилями создаваемых форм и
отчетов. В дополнение к предоставляемым таблицам стилей,
пользователи могут создавать свои собственные стили и сохранять
их для дальнейшего использования.

Новые методы ObjectPAL позволяют импортировать и экспортировать
данные всех поддерживаемых типов непосредственно из программ.
Кроме того, работая с ObjectPAL пользователи могут теперь
посылать в любое приложение среды Windows специальные сообщения
(keystrokes) и управлять диалогом любого другого продукта.

Для работы с пакетами Paradox 4.5 для Windows и Paradox 4.5
Workgroup Edition требуется ПК с процессором 80386 или выше,
Microsoft Windows 3.1 (Enhanced Mode), 4 мегабайта памяти
(рекомендуется 6), винчестер, накопитель для гибких дисков и
устройство "мышь" или другое позиционирующее устройство среды
Windows.

Paradox 4.5 для Windows Development Edition Версия Paradox
Development Edition содержит инструменты разработки, необходимые
для построения и развертывания прикладных программ для систем в
архитектуре клиент/сервер. Эта версия объединяет простоту
использования и мощность пакета Paradox с интегрированными
компонентами разработки и возможностью сопряжения с SQL для
упрощения процесса создания современных прикладных программ.
Среди дополнительных средств - встроенный компилятор,
графический отладчик, визуальные инструменты проектирования,
инструментальные средства для работы с SQL, библиотека
инструментальных средств разработчика и руководство по языку
ObjectPAL.

Paradox 4.5 для DOS Новая версия популярной СУБД для DOS
предлагает улучшенные функциональные возможности как конечным
пользователям, так и разработчикам приложений. Среди добавлений,
рассчитанных на конечного пользователя - расширенная поддержка
"мыши" при создании форм и отчетов. Реализована  возможность
работы с файлами и объектами Paradox, находящимися в разных
каталогах.

Новые команды языка PAL расширяют модель управляемого событиями
программирования. Новый полноэкранный отладчик и более 70 новых
команд языка позволяют разработчикам создавать и
усовершенствовать прикладные программы для среды DOS как в
масштабе отделов, так и для корпораций в целом. Что касается
пользователей, планирующих переход на платформу Windows, Paradox
для DOS по-прежнему является единственной СУБД, обеспечивающей
стопроцентную совместимость со своей Windows-версией - Paradox
для Windows, ведущей реляционной СУБД для этой среды.
