Archive Name: souper10.zip
Description: Extract mail and news to SOUP format (req. IBM TCP/IP)
Operating System/Version: OS/2 3.x
Program Source: cthuang@io.org (Chin Huang)
Proposed directory for placement: /os2/32bit/network

Souper is a network mail and news client program for OS/2 Warp's Internet
Access Kit and IBM TCP/IP.  It creates a SOUP packet from mail and news
fetched from a POP3 server and NNTP server respectively.  It can also send
messages in SOUP reply packets.

The program requires the emx run-time DLLs.  You can get these from

     ftp-os2.cdrom.com:/os2/32bit/unix/emx09a/emxrt.zip
