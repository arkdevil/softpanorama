Below Zero OS/2 Products Catalogue.

Contained in this package are the files required to view our OS/2 catalogue
using the 'VIEW' system contained in OS/2.  To view the catalogue, simply
switch to the directory containing all of the files and execute 'BelowZro'.
This will install the information file and create a Workplace Shell object.
Simply execute the 'Below Zero' icon to view the catalogue.
Further instructions are available within the information file.  If you do 
not have OS/2 loaded you may contact us for a DOS copy.  If you have OS/2
and wish a paper copy, simply use the print system contained in 'VIEW'.
Please contact us if you have any problems or questions.
