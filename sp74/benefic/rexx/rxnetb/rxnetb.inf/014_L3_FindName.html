NETBIOS( 'FindName', adapter, name, stemname)

Adapter is '0' or '1'

name is a 16 byte Netbios name string

returns a string of format NN

nn

    netbios return code

If nn is 0 then stemname. contains the following information

stemname.0

    Count of systems responding the Findname request, list starts at stemname.2
stemname.1
    either 'GroupName' or 'UniqueName'
stemname.2-n (stemname.0 count)
    a binary string that contains the data returned from the find name request.
