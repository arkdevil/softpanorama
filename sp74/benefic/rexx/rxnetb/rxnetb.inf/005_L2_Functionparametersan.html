The NETBIOS function package provides a single function name to be used with a set of parameters. The function name is the first string you used when you RXFUNDADDed the the function, and defaults to NETBIOS.

The general format of the NETBIOS function invocation is: 

NETBIOS( function-string [ parameters ] )

or

call NETBIOS function-string [ parameters ]
