This function creates an OS/2 named pipe. 

Syntax:

  dosrc = RxCreateNPipe( rexxvar, pipename, openmode, pipemode,
                         instance_count, outbufsize, inbufsize,
                         timeout)

where

   rexxvar  = name of Rexx variable into which read/write handle is placed
   pipename = name of pipe

   OpenMode Values
     W    = WriteBehind
     w    = NoWriteBehind
     I    = Inherit
     i    = NoInherit
     N(n) = Access Inbound
     O(o) = Access Outbound
     D(d) = Access Duplex

   PipeMode Values
     W    = Wait
     w    = NoWait
     T    = Pipe Type "Byte"
     t    = Pipe Type "Message"
     R    = Readmode "Byte"
     r    = Readmode "Message"

   Instance_Count
     Integer

   Outbuf Size
     Integer

   Inbuf Size
     Integer

   Timeout
     Integer

  dosrc   = return code from DosCreateNPipe()
