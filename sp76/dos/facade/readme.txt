
  Facade(tm)
  -----------
 
  developers' pcx splashscreen display for DOS applications

  v1.2





  Facade.exe and the sample files
  --------------------------------
  To run the files included in this package (FACADE.EXE and 2 sample files:
  _ADMIN.PCX, _OVERL.EXE) simply execute

                           FACADE <enter>

  from the DOS prompt.  For tolerable viewing results, do not run the utility
  from a floppy drive.

  To install the program, simply copy the files in this package to the hard
  disk directory of your choice.  Please peruse the documentation before
  attempting to evaluate this package.


  Program documentation:  FACADE.TXT


  V1.2 notes:  Clipper developers please take note of the Clipper-specific
  comments under the section Troubleshooting in the documentation.
