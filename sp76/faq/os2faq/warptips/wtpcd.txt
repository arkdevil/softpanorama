TITLE: 950125 IBM THINKPAD 755CD OS/2 WARP ACCESSING THE IDE CDROM              
              AFTER DISKS BASE INSTALLATION                                     
                                                                                
DESCRIPTION OF SYMPTOMS:                                                        
 On this IBM ThinkPad 755CD, the ide CDROM can be swap with the                 
 floppy drive.  In this case, user has taken out the CDROM and puts in          
 the floppy drive to install WARP from diskettes.  After installation,          
 the user's then takes the floppy drive out and put in the CDROM.               
 When the user boots up the system, there is no CDROM access, no CDROM          
 icon available under the DRIVE folder.                                         
                                                                                
RESOLUTION:                                                                     
 Edit the CONFIG.SYS and adds in the following parameters for                   
 the BASEDEV=IBM1S506.ADD:                                                      
 BASEDEV=IBM1S506.ADD /A:0 /U:1 /ATAPI                                          
                                                                                
 Also, adds the following line to the CONFIG.SYS:                               
 BASEDEV=IBMIDECD.FLT                                                           
                                                                                
 Save and exit.  Shutdown the system and reboot.                                
                                                                                
 User should be able to access the cdrom drive.
