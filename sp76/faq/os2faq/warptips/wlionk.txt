TITLE: 950131 LION KING CDROM FROM DISNEY IS INCOMPATIBLE WITH OS2
 
DESCRIPTION OF SYMPTOMS:
    When trying to install the application customer may get an error
    message similiar to:
    Dshell casued a general protection fault in module _MSTEST.EXE 0004:
    _MSTEST.EXE AT 0004:0014 or
    DSHELL caused a stack fault in module _MSTEST.EXE at 0004:2D10
 
SUMMARY OF RESOLUTION:
    The Lion King CD wil not run under any version of OS/2.
 
DETAILS OF RESOLUTION:
    The Lion King CD is attempting to load VXD's into RING 0 and
    OS/2 will not permit that. The CD cannot run with OS/2.
