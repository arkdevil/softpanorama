TITLE: 950127 LOTUS 123 V5.0 FAILS SEAMLESSLY
 
DESCRIPTION OF SYMPTOMS:
    Lotus 123 v5.0 fails to launch seamlessly or in a full
    screen from the OS/2 desktop unless another seamless
    application is started first.
 
SUMMARY OF RESOLUTION:
    Remove the working directory parameters in the Lotus 123
    object settings.
 
DETAILS OF RESOLUTION:
    1.  Open the Lotus 123 v5.0 object settings.
    2.  CHange the working directory parameters to x:\.
         Where x=the root directory of the OS/2 dirve.
    3.  Close the Settings notebook.
 
 
    Lotus 123 will now launch from the Desktop.
