Description
-----------
An essential tool for documenting Visual Basic (Windows) source code.  
It catalogues, displays and prints procedures, types, global variables 
and constants with detailed catalogue entries extracted from remarks 
in the original source.

Can be used with all the sources in a project or all the re-
usable sources that are in the development directory to provide
a catalogue of re-useable code.

Can also create a calling tree for a procedure.  Includes
extensive on-line help, screen and printer configuration facilities.

Requirements
------------
Works only with source code for VB(windows) version 2 or later.
Needs VB(Windows)3 runtime and Windows 3.1.
