*** Installation ***

Follow this easy one-time installation procedure:

1) Create a directory to hold the DirMatch related files.  The
recommended directory name (used in the following instructions) is
"C:\DirMatch".

2) Copy all DirMatch-related files to this directory.  At a minimum
you need the following files:  DirMatch.EXE and DirMatch.HLP.

3) Copy Ctrl3D.DLL to your Windows SYSTEM directory

4) Double click on C:\DIRMATCH\DIRMATCH.EXE in the File Manager.

That's all there is installing DirMatch.  

*** Requirements ***

This program requires Windows 3.1 or Windows for Workgroups.  Some
optional features require external programs.  See the documentation 
for details.
