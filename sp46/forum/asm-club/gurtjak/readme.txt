  Эта программа - драйвер дискетты для тех AT компьютеров,
на которых не читаются дискеты на 720К даже под 800, fdread и т.д.

  Программа представляет собой монитор Int40h, дизассемблированный
из Phoenix BIOS и программа полностью на уровне портов контроллирует
работу с дискетой. Программа не гарантирует чтение 720К дискет, она
только "заменяет собой" ошибочный BIOS, который не дает возможности
этого, т.е. для работы с такими дискетами надо запускать еще
драйвер поддержки fdread, 800 и др.
  Программу рекомендуется запускать до всех других дисковых драйверов.

  Данную программу можно рекомендовать и для тех компьютеров, где
BIOS выделывает с дискетами другие фокусы - например сильно медленная 
работа с диском и др.
