
       LOOKFOR.EXE - утилита  для поиска файлов на диске  и в архивах
                            (  Версия 1.0  )

  **********************************************************************

   Программа LOOKFOR аналогична  известной утилите WHERE, но отличается
тем, что производит поиск файлов также и в архивах (.ARC .ARJ .ICE .LZH
.ZIP ). Кроме того, LOOKFOR определяет, на каких дисках производить
поиск. Чтобы не утомлять Вас долгим описанием всех вариантов запуска
программы , приведу несколько примеров:

 C> lookfor file
      - поиск файла FILE.*  на всех жестких ( и виртуальных) дисках

 В> lookfor file
    -  поиск файла FILE.*  будет проведен только на диске В:

 C> lookfor \file
     - поиск файла на диске C:


 C> lookfor .\.com
                    или
 C> lookfor . .com
     - поиск COM-файлов на диске C:  в текущей директории и ниже


 C> lookfor file another onemore E:
     - поиск файлов FILE.* ANOTHER.* onemore.* на дисках C: и Е:


 C> lookfor E: file
     - поиск файла FILE.* на диске E:

  **********************************************************************

    Версия 1.0 программы LOOKFOR  воспринимает три опции:

      /1 - найти первый подходящий файл и выйти в ДОС
      /u - не искать файлы в архивах (для ускорения поиска)
      /m - повторять поиск на флоппи-дисках. С помощью этой опции
      Вы можете искать требуемый файл на многих дисках, не набирая
      каждый раз команду LOOKFOR.
 
  **********************************************************************

            Для пользователей исходного кода программы LOOKFOR

I Получение EXE-файла.

    Для того, чтобы откомпилировать и слинковать исходные тексты программы
необходимы следующие программы:
     TASM  v 2.0  или более поздняя версия
     TLINK v 3.01 или более поздняя версия
     MAKE  v 3.5  или более поздняя версия
  Все эти программы являются продуктами фирмы Борланд Интернэйшенл.
  Для получения файла LOOKFOR.EXE при наличии всех этих программ (и если
к ним есть доступ в переменной среды PATH) достаточно набрать команду
    MAKE -fLOOKFOR

  Если  нет утилиты MAKE требуемой версии, используйте две команды:
   TASM /m *
   TLINK @LOOKFOR.RSP


II Использование исходного кода.

   Если вам необходимо модифицировать исходный код программы LOOKFOR, вы
имеете право сделать это, в случае если новый вариант программы при вызове
без параметров будет среди прочей информации выдавать на экран строку,
содержащую фамилию автора первоначального варианта программы в том виде, как
программа LOOKFOR.EXE делает это сейчас.

  Если вы используете отрывки из исходного кода программы LOOKFOR, отметьте
в комментариях вашей программы имя автора ( Баранов Ю.А.).


III Использование EXE-программы

   Вы можете использовать эту программу как заблагорассудится и давать ее
кому угодно. Ограничениe: любые изменения EXE-кода напрямую запрещены
(приравниваются к инфицированию вирусом).
