      Определение заголовков

      Для определения заголовка поместите  в  первой  позиции
строки символ FF (ASCII 12), за ним индекс для выполнения пе-
реходов по ссылкам и через пробел название  заголовка. Индекс
может содержать до 20 символов, а название - до  80 символов.
Для вставки управляющих символов  можно  пользоватся  их сим-
вольной записью ^Cnn, где nn - шеснадцатеричный код вводимого
символа. При такой записи ^C0C (код FF, ASCII  12)  не  будет
восприниматься как команда определения заголовка.




      Определение ссылок

      Механизм ссылок позволяет  пользователю  быстро перехо-
дить по справочной системе для получения нужной информации об
интересующем его предмете. Формат команды определения ссылок:

      @Имя_Файла^Имя_ссылки[Выделенный текст]

      В имени ссылки недопустимо  использование символов @,[,
],!,^,%,FF (ASCII 12). Имя файла может опущено,  тогда  поиск
этой ссылки будет производится в текущем help-файле.
      Для определения "короткой клавиши" для  ссылки  укажите
первым символом выделенного текста "!"  и  следующий  за  ним
символ будет использоватся как "короткая клавиша".
      Если в обрабатывемом тексте используется символ "@", то
укажите его дважды : @@.



      Выполнение других программ

      Для выполнения других программ  (используя COMMAND.COM)
используется команда :

      @<командная_строка[Выделенный текст]

      После завершения работы программы  экран воссатноалива-
ется и куросор возвращется на прежнее положение.



      Выполнение макросов

      Для выполнения макросов используется команда :

      @>имя_макроса[Выделенный текст]

      Строка имя_макроса включает в себя имя  макрофайла, имя
макроса и параметры. Но ввиду того строка имя_макро - ссылка,
она ограничена по длине 39 символами. После завершения работы
макроса экран  воссатноаливается  и  куросор  возвращется  на
прежнее положение.



      Отображение текущих назначений клавиш в Help-файле

      Multi-Edit позволяет пользователю  изменять  назначения
практически всех функциональных клавиш и поэтому в Help-сист-
ему редактора введена возможность отображения  текущих назна-
чений клавиш. Для ее использования необходимо  выполнить  два
действия :
      1. В подменю Other\Installation and Setup\Key mapping
проверить поле Menu Identifier для того макроса,  клавиши ко-
торого Вы хотите отслеживать. В этом  поле  должно  находится
имя переменной, в которую будет заносится  наименование прис-
военной этому макросу клавиши. Если это поле пустое, то в не-
го необходимо ввести имя переменной для сохранения наименова-
ния клавиши. Имя переменной может быть любым и  содержать  до
20 символов.
      2. В тексте Help сослаться на имя использованной в пре-
дыущем шаге переменной. Формат ссылки :

      ^Gnn"Имя переменной"

      где nn - максимальная для допустимая длина наименования
клавиши, и имя переменной - это тот идентификатор, который Вы
использовали в шаге 1. При определении длины  учтите, ограни-
чители наименования клавиши < и > входят  в  длину наименова-
ния. Если наименование клавиши длиннее чем  указанная  длина,
то оно усекается. Если же оно короче, то оно дополняется про-
болами слева.



      Экранные атрибуты (Управление выводом)

      Аттрибуты текста на экране могут изменятся  при  помощи
команд, начинающихся с символа "^". Ниже приведен список всех
команд управления цветом :

      ^Ann - установить цвет в nn, где nn  - шестнадцатеричный
             код атрибутов цвета.
      ^B   - включить режим повышенной яркости
      ^b   - выключить
      ^K   - установить признак "мигания" символов
      ^k   - сбросить признак "мигания" символов
      ^R   - включить режим реверсного вывода
             (Повторно - выключить)
      ^0   - обычный текст
      ^1   - цвет 1
      ^2   - цвет 2
      ^3   - цвет 3

      Если в обрабатывемом тексте используется символ "^", то
укажите его дважды : ^^.
