When you have a list of files showing (which is most of the time) you can move around the list using the following keys:

Up Arrow

    Move up one file name.
Down Arrow
    Move down one file name.
Page Up
    Move up one screenful of file names.
Page Down
    Move down one screenful of files names.
Home
    Move to the first file name
End
    Move to the last file name.
Alt-[printable char]
    Move to the first file starting with [printable char]. If you continue to type printable characters, FC/2 will look for a file starting with the string you've typed. Your typing is displayed in the status line of the current panel.
Alt-Up
    Move to the previous file name matching the partial name entered using the above.
Alt-Down
    As above but moves to next match.

When the current panel is in Brief mode, The left and right arrow keys move across one column.
