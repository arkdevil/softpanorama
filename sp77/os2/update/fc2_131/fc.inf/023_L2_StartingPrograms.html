There are several ways to start a program using FC/2.

  • Move the highlight bar to an executable file (.exe .com .cmd .bat) and press ENTER.

  • Move the highlight bar to a file which matches a defined Association and press ENTER.

  • Select a program from the User Menu.

  • Enter the program name on the Command Line.

    With all these methods, except the User Menu, you can start the program in a separate session by pressing Alt-Enter instead of just Enter.
