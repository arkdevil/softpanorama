The following are some of the major features of File Commander/2:

o

    Mouse support.
o
    Full HPFS long file name support.
o
    Online help.
o
    Many configuration options.
o
    Built in file viewer with hex view mode.
o
    Built in editor with cut/copy/paste ability using the OS/2 clipboard.
o
    Screen saver.
o
    Tree views for quick and easy directory changes.
o
    Ability to start OS/2 text mode, PM, DOS and Windows applications.
o
    User definable menu.
o
    Easy creation of WPS objects.
o
    Enhanced wildcard handling.
o
    Flexible file name filtering.
o
    Support for 4DOS/4OS2 descriptions
o
    132 column mode support.
