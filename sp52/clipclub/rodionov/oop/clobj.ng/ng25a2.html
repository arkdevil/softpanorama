      New() -> <oObject>

    Создает новый экземпляр объекта данного класса.
    Внутренняя структура объекта в ODK такова:
                    ┌────────────────────────────┐
     [O_OBJ_CLASS]  │ указатель на класс         │
                    ├────────────────────────────┤
     [O_OBJ_DATA ]  │ указатель на массив данных │
                    └────────────────────────────┘
    Массив данных создается длиной, указанной при создании класса.
    Возвращает созданный объект.
