      addVar(<Имя-переменной>[,<Имя метода>]) -> NIL

    Добавление новой экспортируемой переменной в класс-объект.
    Экспортируемая переменная в Clipper 5.01 есть по сути своей
    тот же метод, но представленный в методах дважды: ИМЯ -
    для получения значения и _ИМЯ - для изменения значения.
    Оба имени суть символьные величины.
    Если переменная <имя-переменной> в классе уже есть, происходит
    замещение переменной.
See Also: AddMethod() AllMethods() Method() MethCount() GetMethod() CopyMethods()
