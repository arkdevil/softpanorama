      AllMethods() -> <array> 

     Возвращает указатель на массив методов класса.
     Каждый элемент представлен подмассивом:
       [O_METH_SEL ] - селектор метода
       [O_METH_METH] - имя метода.
     Возвращаются только методы и переменные, указанные при
     описании класса. Методы родителя в массив не попадут.
See Also: AddMethod() AddVar() Method() MethCount() GetMethod() CopyMethods()
