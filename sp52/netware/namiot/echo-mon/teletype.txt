
                 Эхо-монитор в среде NetWare.
                     SHAREWARE - версия.

     Эта программа предназначена для пользователей, работающих
в среде OС NetWare.  Система Message-Teletype позволяет решать
следующие задачи : с одной из машин сети (или с консоли файло-
вого сервера) по сети осуществляется  рассылка  файла.  Каждый
пользователь  сети может в произвольный момент времени подклю-
читься к чтению этой циркулирующей в  сети  информации.  Таким
образом можно,  например, организовать передачу общей информа-
ции при электронных торгах,  в информационных системах и  т.п.
Подобный  подход - хороший способ подключения к вашим системам
удаленных пользователей.
     В состав системы входят следующие файлы :

     1. sendfile.nlm - загружаемый модуль для передачи файла с
консоли  файл-сервера (NetWare v3.11)   Использование:
   load sendfile файл задержка
     здесь второй параметр - имя файла,  третий параметр - за-
держка в миллисекундах между передачей строк. Например :
  load server\user:sendfile.nlm server\exchange:data\f.txt 500

     2. sendfile.exe - утилита для передачи файла с произволь-
ной рабочей станции сети. Использование :
        sendfile файл задержка
     При необходимости прервать передачу файла нажмите ESC.

     3. recfile.exe - утилита для запуска приема сообщений.

  Заметим, что все функционирование осуществляется в рамках
текущего сервера. Пользователи, которые запустили на своих
машинах утилиту recfile будут получать также и обычные сообщения
Novell NetWare.


     Это программное  средство  распространяется  на  условиях
shareware.  Вы  можете  свободно использовать и копировать эту
программу,  если только это не связано  с  получением  коммер-
ческой выгоды. В противном случае необходима регистрация.
     Если Вы заплатите регистрационный взнос, то сможете полу-
чить полную версию, в которой есть возможность целевой посылки
файла - пользователям с указанными именами.  Возможна поставка
исходных текстов программы.
