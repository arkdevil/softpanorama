Options

-h -H -?
print help text
-r
recursive call ( include subdirectories )
-f
add one name filter to filter list.
by default filter list contain:
.c .h .cpp .hpp .cxx .hxx .java .jav
-o
output dir name. (DOC by default)
-fn
include filename to HTML page
EXAMPLES
java AutoDoc -r c:\work\sources
java AutoDoc -r c:\work\sources c:\work\sources2 ");
java AutoDoc -f .BAS");
java AutoDoc -f .FOR -o DOC\EXAM");
