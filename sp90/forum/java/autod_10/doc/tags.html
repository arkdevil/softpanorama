AutoDoc tags

AutoDoc parses special tags that are recognized when they are embedded within a doc comment.

The tags start with an "at" sign (@).

Tags must start at the beginning of a line.

  • @ALSO

  • @ENDTABLE

  • @ITEM

  • @NODE

  • @TABLE

  • @TITLE
