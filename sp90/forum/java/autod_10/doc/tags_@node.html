Tag @NODE

This key contain full path of item. The @NODE tag is the key tag for AutoDoc. This tag determines the documentation tree without parsing source programming language.

Tag @NODE without value means the root node of documentation (start page, file index.html).

If there is a lot of doc items with the same tag @NODE, all its texts will be concatenated to one item.

For example this AutoDoc documentation item has: 

@NODE Tags/@NODE
