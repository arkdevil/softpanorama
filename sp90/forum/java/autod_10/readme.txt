AutoDoc 1.0

AutoDoc is SHAREWARE automatic documentation  generator.
AutoDoc  makes HTML pages. You must to have an  internet
browser for reading. Latest versions of popular browsres
you can get from www.netscape.com and www.microsoft.com.

REQUIREMENTS: You must to have installed Java machine on
your computer. 

AutoDoc is a Java  aplication and You can  use it on any
platform that has Java interpreter ( Windows, OS/2, MAC,
UNIX, ...)

AutoDoc do not parse syntax! You can use  it for  making
documentation of program in ANY programming language.

INSTALATION is NOT required. Just create  new  directory
and unzip all files. 


TO START AutoDoc type:

> Java AutoDoc -h


For futher info see DOC/index.html
