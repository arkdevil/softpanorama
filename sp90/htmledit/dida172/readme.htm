DiDa Quick Guide

Shareware Message in Previewer

You are seeing this message because DiDa is the evaluation version of DiDaPro HTML Editor.
DiDaPro comes with full preview function without the shareware reminder, plus many other functions. Upon registration you will also receive both the 16bit version (Windows 3.x) and 32bit version (Windows 95) of DiDaPro.
You can disable this message simply by un-check [Options] | [Show Images in Preview]

Setup DiDa Window

If DiDa's Window is too small for you, it can be resized. If you don't like DiDa's default font, simply change it in "Options". After you make those changes you can chose [Save Settings] from the "Options" menu.

Auto Line Select

To select the current line automatically for formatting, hold the [shift] key down, then press the right mouse button.

Change the Default Template

You can replace the content of NORMAL.TXT with any HTML code, then use it as your default HTML template.

Remove Tags

DiDa can automatically remove HTML tags if the highlighted text is preceeded by, and ends with a HTML TAG. DiDa can remove tags shorter than 5 characters, while DiDaPro adds removal regardless of tag size.

Start DiDa in Viewer Mode

If you run DiDa with a filename as its parameter, DiDa will load the file automatically at startup and switch itself to viewer mode.

Display GIF/JPG in Previewer

If GIF/JPGs are not displaying in previewer, please first make sure the [Show Image in Previewer] option is checked under the [OPTIONS] menu. Some Windows' systems require all .DLL files to be located in the \WINDOWS\SYSTEM directory. Simply copy IMLIB221.DLL to there and see if it fixes the problem.

Make DiDa your Default Source Viewer in Netscape

Just go to OPTIONS | GENERAL PREFERENCES | APPS: and set the location of Dida.exe.

Refresh your document

The [HOME] button in previewer also functions as a [REFRESH] button. This will reload your current HTML file into the previewer.
