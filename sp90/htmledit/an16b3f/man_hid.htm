Hidden Features

Click the right mouse button

The right mouse button gives acces to some special functions when clicked at the:

  • editor: an editormenu.
  • palette part of the statusbar: a configuration menu. By the way, you'll need to drag this palette panel open with the >> drawer.
  • filename part of the statusbar: projectsmenu for fast switching.
  • user defined tagbuttons: for defining the button.
