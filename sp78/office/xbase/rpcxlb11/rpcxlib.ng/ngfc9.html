 R_IsMCGA() 
 Determines if a MCGA card is present
──────────────────────────────────────────────────────────────────────────────

 Syntax
        lIsMCGA := R_IsMCGA ()

 Parameters
        (None)

 Returns
        lIsMCGA         .T. :   MCGA card present or can be emulated
                        .F. :   No MCGA card present

 Description
        This function can determine if the PC supports MCGA.

 Example
        See PCXDemo.PRG
