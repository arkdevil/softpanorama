.topic SubMenu = 1000
 
  ▌TMenuBar▐

TMenuBar objects represent the horizontal menu bars from which menu
selections can be made by:

 ■ direct click
 ■ F10 select and shortcut keys
 ■ select (highlight) and press Enter
 ■ hot keys

.topic Item
  
Represents a menu item selections can be made by:

 ■ direct click
 ■ select (highlight) and press Enter
 ■ hot keys


.topic InputLine

  A TInputLine object provides a basic input
line string editor.

.topic NumInputLine

  A TNumInputLine object provides a input
line numeric editor.

.topic TListViewer
  TListViewer is essentially a base type from which
to derive list viewers of various kinds, such as TListBox.

.topic OkButton

 ┌────────┐
 │[  OK  ]│
 └────────┘

Press Enter or click the OK button to put this
message window away and get into the
environment.


.topic cancel Button

 ┌────────────┐
 │ [ Cancel ] │
 └────────────┘

If you choose Cancel, nothing changes and no
action occurs, but the dialog box is put away.

Esc is always a keyboard shortcut for Cancel
even if a Cancel button does not appear.
